Pendahuluan

Definisi

CRUD adalah singkatan dari create read update delete. platform yang digunakan adalah website. Website adalah sebuah kumpulan dari beberapa halaman yang saling terhubung dan berisi beberapa fitur dan juga informasi.

Website ini bisa kita gunakan untuk memproses berbagai data dengan berbagai fitur yang kita inginkan. Seseorang web developer atau pengembang website ketika pertama kali belajar pasti yang akan dilakukan adalah mempelajari konsep dasar dari CRUD. Ketika kita nanti akan mengembangkan program CRUD, kita memiliki 3 komponen dasar yang perlu kita pahami, yaitu

  • Data

    digunakan untuk berbagai macam proses

  • Interface

    atau tampilan yang bisa dilihat atau diedit untuk memproses suatu data

  • Proses

    pengolahan dari data yang kita miliki dengan menggunakan interface yang sudah disediakan

Penjelasan

1. Data

Data adalah sekumpulan fakta yang belum memiliki fungsi. nantinya data ini akan diolah menjadi informasi

  • Database

    tempat penyimpanan data

  • MySQL

    aplikasi yang digunakan untuk mengoperasikan database


2. Interface

Interface adalah sebuah tampilan yang mendeskripsikan apapun yang kita perlukan dalam mengakses data. Dalam tanda kutip tampilan, tampilan yang hanya tampil dan belum bisa memproses apapun. Untuk membuat tampilan atau interface kita akan membutuhkan

  • HTML

    kerangka dasar dalam membuat sebuah website

  • CSS

    digunakan untuk mempercantik tampilan interface

  • JavaScript

    untuk membuat interface lebih interaktif

  • Bootstrap 5

    kerangka kerja css dan js yang disediakan browser


3. Proses

Setelah punya interface dan data tetapi masih belum terhubung dan belum bisa digunakan, maka kita memerlukan sebuah proses agar data yang ada di dalam database kita bisa diakses melalui interface yang kita miliki.

                    echo $query;
                }

                $id_siswa = $_POST['id_siswa'];
                $nisn = $_POST['nisn'];

                $jenis_kelamin = $_POST['jenis_kelamin'];

                } else {
                    $foto = $_FILES['foto_siswa']['name'];
