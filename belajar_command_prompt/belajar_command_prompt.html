Command Prompt Atau Terminal

Command Prompt Codes

Ketik `ls` pada command prompt atau terminal.

Ketik `pwd` pada command prompt atau terminal.

Ketik `cd` diikuti dengan nama folder untuk masuk kedalam folder tersebut. Contohnya, `cd documents` untuk masuk kedalam folder "documents".

Ketik `cd ..` untuk keluar dari folder.

Ketik `mkdir` diikuti dengan nama folder baru yang ingin dibuat. Contohnya, `mkdir NewFolder` akan membuat folder baru dengan nama "NewFolder".

Ketik `rmdir` diikuti dengan nama folder yang ingin dihapus. Contohnya, `rmdir NewFolder` akan menghapus folder dengan nama "NewFolder".

Ketik `mv` diikuti dengan nama folder lama, dan diikuti dengan nama folder baru. Contohnya, `mv docs documents` akan mengubah nama folder dengan nama "docs" menjadi "documents".

Ketik `touch` diikuti dengan nama file yang ingin dibuat. Contohnya, `touch NewFile.txt` akan membuat file baru dengan nama "NewFile.txt".

Ketik `rm` diikuti dengan nama file yang ingin dihapus. Contohnya, `rm NewFile.txt` akan menghapus file dengan nama "NewFile.txt".

Ketik `mv` diikuti dengan nama file lama, dan diikuti dengan nama file baru. Contohnya, `mv FileLama.txt FileBaru.txt` akan mengubah nama file dengan nama "FileLama.txt" menjadi "FileBaru.txt".

Ketik `echo "Hello World" > NewFile.txt` untuk menulis teks. perhatikan tanda `>>` untuk menambah teks dan tanda `>` untuk menimpa teks. Jika ada file dengan nama tersebut maka teks akan disimpan di dalamnya, dan jika tidak ada maka akan dibuatkan

Ketik `cat` diikuti dengan nama file yang ingin ditampilkan. Contohnya, `cat NewFile.txt` untuk menampilkan isi dari file "NewFile.txt".

Ketik `cp` diikuti dengan nama file atau folder yang ingin disalin, lalu diikuti dengan lokasi tujuan. Contohnya, `cp NewFile.txt /home/user/Documents/` akan menyalin file "NewFile.txt" ke folder "Documents" di dalam folder "user".

Ketik `mv` diikuti dengan nama file atau folder yang ingin dipindahkan, lalu diikuti dengan lokasi tujuan. Contohnya, `mv NewFile.txt /home/user/Documents/` akan memindahkan file "NewFile.txt" ke folder "Documents" di dalam folder "user".

Pengenalan Batch File

Apa Itu Batch File

Batch file adalah sebuah file teks yang berisi serangkaian perintah yang dapat dieksekusi secara otomatis oleh sistem operasi. Batch file digunakan untuk menyusun dan mengatur serangkaian tugas yang harus dilakukan secara berurutan atau dalam satu waktu tertentu. Biasanya, batch file digunakan untuk otomatisasi tugas-tugas rutin dalam lingkungan Windows, seperti menjalankan beberapa perintah sekaligus, mengatur pengaturan sistem, menjalankan program atau skrip, serta mengelola file dan folder. Dengan menggunakan batch file, pengguna dapat menghemat waktu dan usaha yang seharusnya diperlukan untuk menjalankan setiap perintah secara manual.

Apa Fungsi Batch File

Batch file memiliki beberapa fungsi penting, antara lain

1. Otomatisasi Tugas
: Batch file digunakan untuk mengotomatisasi tugas-tugas rutin seperti menjalankan serangkaian perintah atau program secara berurutan. Dengan batch file, pengguna dapat menghindari keharusan mengetik dan menjalankan perintah secara manual setiap kali.

2. Pengaturan Sistem
: Batch file memungkinkan pengguna untuk mengatur pengaturan sistem secara efisien. Misalnya, pengguna dapat menggunakan batch file untuk mengubah pengaturan jaringan, mengonfigurasi perangkat keras, mengelola layanan, dan melakukan tugas-tugas administratif lainnya.

3. Penjadwalan Tugas
: Batch file memungkinkan penjadwalan tugas untuk dijalankan pada waktu yang ditentukan. Pengguna dapat membuat batch file dan menggunakan alat bantu sistem operasi seperti Task Scheduler untuk menjalankan batch file tersebut secara otomatis pada waktu yang ditentukan.

4. Penjadwalan Tugas
: Batch file memungkinkan penjadwalan tugas untuk dijalankan pada waktu yang ditentukan. Pengguna dapat membuat batch file dan menggunakan alat bantu sistem operasi seperti Task Scheduler untuk menjalankan batch file tersebut secara otomatis pada waktu yang ditentukan.

5. Pengolahan File dan Folder
: Batch file dapat digunakan untuk mengelola file dan folder. Pengguna dapat membuat batch file untuk menggabungkan, memindahkan, menghapus, atau mengubah nama file dan folder dengan cepat dan efisien.

6. Eksekusi Program dan Skrip
: Batch file memungkinkan pengguna untuk menjalankan program atau skrip secara otomatis. Ini memungkinkan pengguna untuk menjalankan serangkaian perintah atau skrip yang kompleks dengan sekali klik atau melalui penjadwalan tugas.

Dengan berbagai fungsi ini, batch file menjadi alat yang kuat untuk mengotomatisasi tugas-tugas dalam sistem operasi Windows, meningkatkan efisiensi, dan menghemat waktu.

Apa Yang Harus Dipelajari

Untuk memahami tentang batch file, ada beberapa hal yang harus dipelajari

1. Dasar-Dasar Pemrograman
: Memahami konsep dasar pemrograman seperti variabel, percabangan, pengulangan, dan fungsi sangat penting dalam menguasai batch file. Anda perlu memahami bagaimana perintah-perintah dalam batch file dieksekusi dan bagaimana mengelola aliran eksekusi dengan menggunakan kontrol logika.

2. Perintah Batch
: Batch file menggunakan sejumlah perintah bawaan yang disediakan oleh sistem operasi. Anda perlu mempelajari perintah-perintah ini, termasuk sintaksisnya, opsi-opsi yang tersedia, dan cara menggunakannya. Contoh beberapa perintah yang umum digunakan adalah echo, set, if, for, call, dan pause.

3. Variabel Lingkungan
: Batch file menggunakan variabel lingkungan untuk menyimpan dan mengelola data. Anda perlu memahami cara mendefinisikan variabel, memberikan nilai, dan mengaksesnya dalam batch file. Selain itu, juga penting untuk memahami variabel lingkungan bawaan yang disediakan oleh sistem operasi, seperti %USERNAME%, %DATE%, dan %TIME%.

4. Pengolahan File dan Folder
: Salah satu fungsi utama batch file adalah pengolahan file dan folder. Anda perlu mempelajari perintah-perintah yang digunakan untuk mengelola file dan folder, seperti copy, move, del, dir, dan ren. Selain itu, pemahaman tentang wildcard dan cara menggunakannya dalam batch file juga diperlukan.

5. Penjadwalan Tugas
: Batch file dapat digunakan untuk menjalankan tugas secara terjadwal. Anda perlu mempelajari cara menggunakan alat bantu sistem operasi, seperti Task Scheduler di Windows, untuk menjadwalkan dan menjalankan batch file pada waktu yang ditentukan.

Selain itu, praktik secara aktif membuat dan menjalankan batch file juga akan membantu meningkatkan pemahaman Anda tentang konsep dan penggunaan batch file. Melalui eksperimen dan pengalaman, Anda dapat mengembangkan keterampilan dalam membuat batch file yang efektif dan efisien.

Percabangan Batch File

Percabangan Pada Batch File

Percabangan adalah sebuah konstruksi yang digunakan untuk membuat keputusan atau melakukan perintah berdasarkan kondisi yang diberikan. Struktur dasar percabangan dalam batch file adalah menggunakan perintah IF dan ELSE. Berikut adalah contoh percabangan pada batch file

baca selengkapnya
1. IF Statement

Pernyataan IF memungkinkan Anda untuk menjalankan serangkaian perintah jika kondisi tertentu terpenuhi. Contohnya

IF
                                    
    IF %nama_variabel% == 1 (
        perintah
    )
IF - ELSE
                                    
    IF %nama_variabel% == 1 (
        perintah 1
    ) ELSE (
        perintah 2
    )
IF - ELSE IF - ELSE
                                    
    IF %nama_variabel% == 1 (
        Perintah 1
    ) ELSE IF %nama_variabel% == 2 (
        perintah 2
    ) ELSE (
        perintah 3
    )

Anda dapat menggunakan operator perbandingan lain seperti == (sama dengan), NEQ (tidak sama dengan), LSS (kurang dari), LEQ (kurang dari atau sama dengan), GTR (lebih besar dari), GEQ (lebih besar dari atau sama dengan) untuk membandingkan nilai variabel dengan nilai yang diinginkan.

2. GOTO Statement

Pernyataan GOTO memungkinkan Anda untuk melompat ke label tertentu dalam batch file. Contohnya

                    
    set /p pilihan=Masukkan pilihan (1 atau 2): 

    IF %pilihan% == 1 (
        goto option1
    ) ELSE IF %pilihan% == 2 (
        goto option2
    ) ELSE (
        goto invalid
    )

    :option1
    echo Anda memilih opsi 1
    goto end

    :option2
    echo Anda memilih opsi 2
    goto end

    :invalid
    echo Pilihan tidak valid.
    goto end

    :end
    echo Selesai.

Jika anda memasukan angka 1 maka label "option1" dan perintahnya akan dijalankan. Jika anda memasukan angka 2 maka label "option2" dan perintahnya akan dijalankan. Dan jika anda memasukan pilihan selain angka 1 dan 2 maka label "invalid" dan perintahnya akan dijalankan.

3. CHOICE statement

Pernyataan CHOICE memungkinkan Anda untuk menampilkan pilihan kepada pengguna dan menjalankan perintah berdasarkan pilihan yang dipilih. Contohnya

                    
    echo Pilih bahasa yang ingin Anda gunakan:
    echo 1. Bahasa Inggris
    echo 2. Bahasa Spanyol
    echo 3. Bahasa Prancis

    CHOICE /C 123 /M "Masukkan pilihan Anda:"

    IF %ERRORLEVEL% EQU 1 (
        echo Anda memilih Bahasa Inggris.
    ) ELSE IF %ERRORLEVEL% EQU 2 (
        echo Usted ha elegido el idioma español. (Anda telah memilih Bahasa Spanyol.)
    ) ELSE IF %ERRORLEVEL% EQU 3 (
        echo Vous avez choisi la langue française. (Anda telah memilih Bahasa Prancis.)
    ) ELSE (
        echo Pilihan tidak valid.
    )

Pernyataan CHOICE dengan opsi /C 123 digunakan untuk menentukan pilihan yang valid, yaitu 1, 2, atau 3. Pesan "/M" menampilkan pesan "Masukkan pilihan Anda:".

4. Existence of Files/Folders

Anda juga dapat menggunakan pernyataan pengkondisian untuk memeriksa keberadaan file atau folder. Contohnya

                                    
    IF EXIST "C:\path\to\file.txt" (
        echo File.txt ditemukan.
    ) else (
        echo File.txt tidak ditemukan.
    )

Dalam contoh ini, pernyataan IF digunakan untuk memeriksa apakah file "file.txt" ada atau tidak pada path yang ditentukan. Jika file tersebut ada, perintah di dalam blok IF akan dieksekusi dan pesan "File.txt ditemukan" akan ditampilkan. Jika file tidak ada, perintah di dalam blok ELSE akan dieksekusi dan pesan "File.txt tidak ditemukan" akan ditampilkan.

Pengulangan Batch File

Pengulangan Pada Batch File

Perulangan pada batch file adalah kemampuan untuk menjalankan serangkaian perintah berulang kali. Dalam batch file, ada beberapa cara untuk melakukan perulangan. Dua jenis perulangan yang umum digunakan adalah perulangan FOR dan perulangan DO - WHILE.

baca selengkapnya
1. FOR

Perulangan FOR digunakan untuk mengulangi serangkaian perintah untuk setiap item dalam satu atau lebih daftar item.

                                    
    FOR %%i in (1 2 3 4) do (
        echo %%i
    )

Dari contoh diatas, "%%i" adalah variabel index, dia ditulis dengan 2 tanda persen dan 1 karakter. Selanjutnya "(1 2 3 4)" adalah daftar yang akan ditampilkan, setiap daftar dipisahkan dengan spasi

2. DO - WHILE

Perulangan ini digunakan untuk mengulangi serangkaian perintah selama kondisi tertentu terpenuhi dan akan terus dijalankan sampai kondisi yang ditentukan tidak lagi terpenuhi.

                                    
    set /a counter=1

    :loop
        echo %counter%
        set /a counter+=1
        if %counter% LEQ 5 (
            goto loop
        )

Dari contoh diatas, variabel "counter" memiliki value 1. Di dalam loop, variabel "counter" akan di tambah satu pada setiap pengulangannya. Dan variabel "counter" akan di ulang selama dia kurang dari sama dengan 5.

Anda bisa mempelajari lebih lanjut tentang pengulangan dengan masuk ke CMD dan ketikan perintah "help for", disitu terdapat banyak contoh penggunaan pengulangan dan juga parameter tambahan.

Parameter Fungsi
/D If set contains wildcards, then specifies to match against directory names instead of file names.
/R Walks the directory tree rooted at [drive:]path, executing the FOR statement in each directory of the tree. If no directory specification is specified after /R then the current directory is assumed. If set is just a single period (.) character then it will just enumerate the directory tree.
/L The set is a sequence of numbers from start to end, by step amount. So (1,1,5) would generate the sequence 1 2 3 4 5 and (5,-1,1) would generate the sequence (5 4 3 2 1)

        for /L %%i in (1,1,8) do (
            echo Mata Kuliah  : !mata_kuliah[%%i]!

            set /p nilai[%%i]="Nilai        : "
            echo.
            
            if !nilai[%%i]! geq 81 ( if !nilai[%%i]! leq 100 ( set grade[%%i]=A  & set bobot[%%i]=4.00 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] )) 
            if !nilai[%%i]! geq 76 ( if !nilai[%%i]! leq 80  ( set grade[%%i]=AB & set bobot[%%i]=3.50 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 71 ( if !nilai[%%i]! leq 75  ( set grade[%%i]=B  & set bobot[%%i]=3.00 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 66 ( if !nilai[%%i]! leq 70  ( set grade[%%i]=BC & set bobot[%%i]=2.50 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 61 ( if !nilai[%%i]! leq 65  ( set grade[%%i]=C  & set bobot[%%i]=2.00 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 56 ( if !nilai[%%i]! leq 60  ( set grade[%%i]=CD & set bobot[%%i]=1.50 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 51 ( if !nilai[%%i]! leq 55  ( set grade[%%i]=D  & set bobot[%%i]=1.00 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))
            if !nilai[%%i]! geq 0  ( if !nilai[%%i]! leq 50  ( set grade[%%i]=E  & set bobot[%%i]=0.00 & set /a total_bobot[%%i]=sks[%%i]*bobot[%%i] ))

            set /a total_sks+=sks[%%i]
            set /a total_bobot+=total_bobot[%%i]
            
        )

        cls
        echo. >> %nim%_%nama%.txt
        echo [==================================================================] >> %nim%_%nama%.txt
        echo    Mata Kuliah                     SKS     Nilai   Grade   Bobot     >> %nim%_%nama%.txt
        echo [==================================================================] >> %nim%_%nama%.txt
        echo. >> %nim%_%nama%.txt

            echo    !mata_kuliah[%%i]!    !sks[%%i]!        !nilai[%%i]!      !grade[%%i]!   !bobot[%%i]! >> %nim%_%nama%.txt

        echo [==================================================================] >> %nim%_%nama%.txt
        echo. >> %nim%_%nama%.txt

        echo    NIM          :  %nim%          >> %nim%_%nama%.txt
        echo    Nama         :  %nama%         >> %nim%_%nama%.txt
        echo    Kelas        :  %kelas%        >> %nim%_%nama%.txt
        echo    Prodi        :  %prodi%        >> %nim%_%nama%.txt
        echo    Total SKS    :  %total_sks%    >> %nim%_%nama%.txt
        echo    Total Bobot  :  %total_bobot%  >> %nim%_%nama%.txt

        set /a ip=%total_bobot%/%total_sks%
        echo    IP           :  %ip%           >> %nim%_%nama%.txt

        echo Data Berhasil Disimpan . . .
        echo.
        pause

        type %nim%_%nama%.txt
        echo.
