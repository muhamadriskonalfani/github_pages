CSS Pendahuluan

Definisi

CSS adalah singkatan dari Cascading Style Sheet yang digunakan untuk memberikan style seperti warna, ukuran, dan posisi ke dalam elemen-elemen HTML.

  • Aturan yang digunakan untuk menampilkan elemen / tag HTML
  • Dibuat terpisah dengan HTML
  • Bertujuan untuk memisahkan konten dan style
  • 1 CSS dapat digunakan untuk banyak halaman HTML
  • 1 Halaman HTML dapat terlihat berbeda jika menggunakan CSS yang berbeda pula

Anatomi CSS

CSS memiliki anatomi atau struktur sebagai berikut 

selector { property: value; }
Berikut penjelasanya

  • Selector
    adalah sebuah bagian dari CSS yang digunakan untuk memilih dan memanipulasi sebuah elemen spesifik yang ada di HTML. Elemen HTML dipilih berdasarkan tag, id, class bahkan pola / pattern. Semakin kompleks struktur HTML, selector juga bisa semakin kompleks / spesifik.

  • Property
    adalah sebuah bagian dari CSS yang berupa nama dari style seperti color, font-size, atau background yang akan di berikan kepada selectornya.

  • Value
    adalah isi atau nilai dari sebuah properti, nilai ini yang akan dikirim ke dalam properti, dan properti akan meneruskanya ke selector.

Penempatan CSS

Kita bisa menyimpan CSS di dalam halaman yang sama atau di halaman terpisah dengan ekstensi .css yang nantinya akan dihubungkan dengan halaman HTML.

1. embed

Kita bisa simpan di dalam file yang sama, dengan tag style yang disimpan didalam tag head

< stylx>...< /style>
2. inline

Kita juga bisa menyisipkan atribut style kedalam sebuah elemen HTML

< p style="color: lightblue;">...< /p>
3. external

Kita buat CSS terpisah dengan halaman HTML dengan ekstensi .css dan dihubungkan dengan halaman HTML menggunakan tag link didalam tag head

< link rel="stylesheet" href="style.css">

CSS Font Styling

Definisi

Font Styling adalah bagaimana kita memberikan style pada huruf yang ada pada elemen-elemen HTML. Perbedaan antara properti font dengan text pada CSS adalah jika font itu terkait dengan tulisanya atau type-face nya sedangkan text itu terkait dengan properti pendukung.

Font Styling

Properti yang terkait dengan font styling

  • font-family

    mengatur jenis font yang digunakan

  • font-size

    mengatur ukuran font

  • font-weight

    mengatur ketebalan font

  • font-variant

    mengubah font menjadi small caps

  • font-style

    memberikan style pada font seperti membuat teks menjadi miring

  • line-height

    mengatur spasi antar baris

Implementasi

1. font-family

Berfungsi untuk mengatur jenis font yang digunakan. font-family memiliki 2 value, ada nama font dan generic family.

Penulisan
        
    selector {
        font-family: nama_font, generic_family;
    }
Contoh
        
    body {
        font-family: "Times New Roman", serif;
    }

    p {
        font-family: arial, verdana, sans-serif;
    }
Bisa memberikan lebih dari satu value. Times New Roman, arial dan verdana adalah nama font-nya, sedangkan serif dan sans-serif adalah generic family.
2. font-size

Berfungsi untuk mengatur ukuran font. valuenya adalah numerik (px).

Penulisan
                                    
    selector {
        font-size: value;
    }
Contoh
                                    
    h1 {
        font-size: 100px;
    }
    
    p {
        font-size: 18px;
    }
Value
  • px
    - ukuran absolute
  • %
    - relatif terhadap ukuran layar
  • em
    - relatif terhadap jenis font
3. font-weight

Berfungsi untuk mengatur ketebalan font.

Penulisan
                                    
    selector {
        font-weight: value;
    }
Contoh
                                    
    h1 {
        font-weight: bold;
    }

    p {
        font-weight: normal;
    }
Value
  • lighter
    - lebih ringan dari normal
  • normal
    - ketebalan normal
  • 100-900
    - disesuaikan dengan angka
  • bold
    - tebal
  • bolder
    - lebih tebal
4. font-variant

Berfungsi untuk mengubah font menjadi small caps.

Penulisan
                                    
    selector {
        font-variant: value;
    }
Contoh
                                    
    h1 {
        font-variant: small-caps;
    }
    
    p {
        font-variant: normal;
    }
Value
  • normal
    - bentuk normal
  • small-caps
    - contoh Hello World
5. font-style

Berfungsi untuk memberikan style pada font seperti membuat teks menjadi miring.

Penulisan
                                    
    selector {
        font-style: value;
    }
Contoh
                                    
    h1 {
        font-style: oblique;
    }

    p {
        font-style: italic;
    }
Value
  • normal
    - style normal
  • italic
    - teks miring
  • oblique
    - teks miring
6. line-height

Berfungsi untuk mengatur spasi antar baris. valuenya adalah numerik (px).

Penulisan
                                    
    selector {
        line-height: value;
    }
Contoh
                                    
    p {
        line-height: 20px;
    }
Valuenya ada 3 jenis
  • normal
    - normal
  • px
    - ukuran absolute
  • em
    - relatif terhadap jenis font

Shorthand

Shorthand digunakan untuk menyingkat beberapa properti CSS dalam satu selector.

Contoh kode
                
    body {
        font-style: italic;
        font-variant: normal;
        font-weight: bold;
        font-size: 16px;
        line-height: 18px;
        font-family: helvetica, arial, sans-serif;
    }
Bisa di singkat menjadi
                
    body {
        font: italic normal bold 16px/18px helvetica, arial, sans-serif;
    }
Perhatikan juga urutanya
  • font-style (optional)
  • font-variant (optional)
  • font-weight (optional)
  • font-size (wajib)
  • line-height (optional)
  • font-family (wajib)

CSS Background

Definisi

Didalam CSS kita bisa memberikan background pada elemen-elemen HTML termasuk pada halamanya sendiri, termasuk body atau pada elemen-elemen yang spesifik.

Background

Properti yang terkait dengan background

  • background-color

    memberi warna pada background

  • background-image

    menambahkan gambar pada sebuah elemen

  • background-position

    mengatur posisi gambar pada background

  • background-repeat

    mengatur jenis pengulangan gambar pada background

Implementasi

1. background-color

Berfungsi untuk memberi warna pada background.

Penulisan
        
    selector {
        background-color: value;
    }
Contoh
        
    body {
        background-color: royalblue;
    }
Value
  • nama warna
    - contoh red, lightseagreen, royalblue, ...
  • hexadecimal
    - contoh #ff0000, #20b2aa, #4169e1, ...
  • rgb
    - contoh rgb(32,178,170), rgb(65,105,225), ...
2. background-image

Berfungsi untuk menambahkan gambar pada sebuah elemen.

Penulisan
        
    selector {
        background-image: url(value);
    }
Contoh
        
    body {
        background-image: url(bg1.jpg);
    }

Valuenya adalah nama gambarnya, jika gambarnya ada didalam folder maka tulis nama folder / nama gambar, jika diluar folder maka tulis .. / nama gambar.

3. background-position

Berfungsi untuk mengatur posisi gambar pada background.

Penulisan
        
    selector {
        background-image: url(bg1.jpg);
        background-position: value;
        width: 100%;
    }
Contoh
        
    body {
        background-image: url(bg1.jpg);
        background-position: top left;
        width: 100%;
    }
Value
  • top left
    - kiri atas
  • top center
    - tengah atas
  • top right
    - kanan atas
  • center left
    - kiri tengah
  • center center
    - tengah
  • center right
    - kanan tengah
  • bottom left
    - kiri bawah
  • bottom center
    - tengah bawah
  • bottom right
    - kanan bawah
  • x% y%
    - sumbu x dan sumbu y
  • x-pos y-pos
    - sumbu x dan sumbu y

Ingat bahwa setiap elemen HTML punya kotak yang ukurannya tergantung pada isi atau kontenya, jika ingin menggunakan properti background-position pastikan untuk menentukan ukuran kotaknya dengan properti width.

4. background-repeat

Berfungsi untuk mengatur jenis pengulangan gambar pada background.

Penulisan
        
    selector {
        background-image: url(bg1.jpg);
        background-repeat: value;
    }
Contoh
        
    body {
        background-image: url(bg1.jpg);
        background-repeat: repeat-x;
    }
Value
  • repeat

    background akan diulang secara default

  • repeat-x

    background akan diulang dari kiri ke kanan (sumbu x)

  • repeat-y

    background akan diulang dari atas ke bawah (sumbu y)

  • no-repeat

    gambar pada background tidak diulang

Untuk mempraktikan background-repeat bisa gunakan gambar yang kecil atau bisa tentukan sendiri ukuran gambarnya.

Shorthand

Shorthand digunakan untuk menyingkat beberapa properti CSS dalam satu selector. Sama dengan properti font, properti background juga memiliki shorthand.

Penulisan
                
    selector {
        background: color url() position repeat;
    }
Contoh
                
    body {
        background-color: royalblue;
        background-image: url(bg1.jpg);
        background-position: top left;
        background-repeat: no-repeat;
    }
Bisa di singkat menjadi
                
    body {
        background: royalblue url(bg1.jpg) top left no-repeat;
    }
Perhatikan juga urutanya
  • background-color
  • background-image
  • background-position
  • background-repeat

Jika ingin menggunakan background-position maka background-repeat harus di kosongkan (set default), begitu juga sebaliknya.

CSS Selector

Definisi

Didalam CSS, selector ini adalah sesuatu yang digunakan oleh CSS untuk mengenali sebuah elemen HTML yang akan kita beri style nantinya.

Selector

Apa saja yang bisa kita gunakan sebagai selector

  • elemen HTML

    seperti h1, p, a, ...

  • id

    atribut spesifik, value dari atribut ini hanya boleh ada satu pada satu halaman

  • class

    atribut yang valuenya boleh ada di beberapa elemen yang diinginkan saja pada sebuah halaman

  • complex selector

    ini digunakan untuk memilih satu atau lebih elemen yang spesifik, seperti memilih salah satu list dari banyak list

Implementasi

1. elemen HTML

Kita bisa menggunakan elemen HTML sebagai selector, seperti body, p, a, dsb.

Selector CSS
        
    body {
        properti: value;
    }

    p {
        properti: value;
    }

    a {
        properti: value;
    }
2. id

Kita juga bisa menggunakan id yang kita buat sendiri di dalam sebuah elemen HTML.

Contoh Kode
                                    
    < section id="satu">
        ...
    < /section>
Selector CSS
        
    section#satu {
        properti: value;
    }
3. class

Kita juga bisa menggunakan class yang kita buat sendiri di dalam sebuah elemen HTML.

Contoh Kode
                                    
    < div class="container">
        ...
    < /div>
Selector CSS
        
    .container {
        properti: value;
    }
4. complex selector

Kita juga bisa menggabungkan elemen HTML, id, dan class menjadi satu selector, bahkan kita juga bisa menggunakan pseudo CSS.

Contoh kode
                                    
    < ul class="list" id="daftar">
        < li>...< /li>
        < li>...< /li>
        < li>...< /li>
    < /ul>
Selector CSS
        
    ul.list#daftar:nth-child(2) {
        properti: value;
    }
Dia akan memilih list kedua dari elemen ul yang memiliki class list dan id daftar.

CSS Pseudo Class

Definisi

Kelas semu yang dimiliki sebuah elemen HTML yang membuat kita dapat mendefinisikan style pada keadaan tertentu dari elemen tersebut.

Pseudo Class

Pseudo Class ini terbagi menjadi beberapa tipe

Pseudo class yang berhubungan dengan link
  • :link

    style default pada sebuah link (a yang memiliki href)

  • :hover

    style ketika kursor mouse berada diatas sebuah link

  • :active

    style ketika sebuah link diklik (keadaan aktif)

  • :visited

    style ketika sebuah link sudah pernah dikunjungi sebelumnya (menggunakan browser yang sama)

Pseudo class yang berhubungan dengan posisi elemen (1)
  • :first-child

    memilih elemen pertama dari sebuah parent (elemen pembungkusnya)

  • :last-child

    memilih elemen terakhir dari sebuah parent (elemen pembungkusnya)

  • :nth-child(n)

    memilih elemen ke-n dari sebuah parent (elemen pembungkusnya)

    n bisa berarti urutan (1), (2), ... atau pola (2n), (3n+2), (4n+1) atau ganjil dan genap (even) & (odd)

Pseudo class yang berhubungan dengan posisi elemen (2)
  • :first-of-type

    memilih elemen pertama dari sebuah jenis / tipe tag

  • :last-of-type

    memilih elemen terakhir dari sebuah jenis / tipe tag

Implementasi

Cara menggunakan pseudo class adalah dengan menuliskan class terlebih dahulu dan diikuti dengan pseudo class.

Penulisan
                
    selector:pseudo-class {
        properti: value;
    }
Contoh
                
    a:hover {
        color: blue;
    }

    ul li:nth-child(2) {
        background-color: red;
    }
elemen a jika disentuh maka teksnya akan berubah menjadi warna biru dan list kedua backgroundnya akan berwarna merah.

CSS Specificity

Definisi

Setiap deklarasi CSS (selector) memiliki berat yang berbeda. Berat tersebut menentukan seberapa spesifik sebuah elemen dapat dipilih oleh selector.

Specificity

Table specificity
specificity nilai
elemen HTML 1
class 10
id 100
Penjelasan
  • Jika ada dua selector yang sama, maka selector terakhir yang akan ditampilkan
  • Jika selectornya berbeda, maka selector yang lebih berat dialah yang akan ditampilkan
  • Satu selector bisa lebih dari satu specificity, semua specificity tersebut bisa dijumlahkan, selanjutnya selector yang lebih berat dialah yang akan ditampilkan

Implementasi

Contoh 1

Contoh semisal kita punya sebuah paragraf dan paragraf tersebut diberi CSS

        
    p { color: blue; }

    p { color: red; }
jika ada dua selector yang sama, beratnya juga sama tapi nilainya berbeda, maka selector yang terakhir yang akan terlihat. Tulisan paragraf berwarna merah.
Contoh 2

Contoh semisal kita punya sebuah paragraf dan paragraf tersebut memiliki atribut class teks

        
    .teks { color: blue; }

    p { color: red; }
jika selectornya berbeda, selector yang lebih berat dialah yang akan ditampilkan. class (.teks) lebih berat dari elemen HTML (p). Tulisan paragraf berwarna biru.
Contoh 3

Contoh semisal kita punya sebuah paragraf dan paragraf tersebut memiliki atribut class teks dan id satu

        
    .teks { color: blue; }

    p.teks#satu { color: red; }

jika selectornya berbeda, selector yang lebih berat dialah yang akan ditampilkan.

p.teks#satu lebih berat dari .teks

CSS Layouting Display

Definisi

Setiap tag pada HTML berada di dalam sebuah kotak. properti display pada CSS mengatur perilaku dari kotak tersebut. setiap tag pada HTML memiliki nilai default untuk properti display. tetapi kita juga dapat mengubah perilaku dari tag tersebut dengan mengganti valuenya.

Display

Ada beberapa value dari properti display

  • inline

    b, strong, i, em, a, span, sub, sup, button, input, label, select, textarea, ...

  • inline-block

    tidak ada elemen dengan properti display inline-block

  • block

    h1-h6, p, ol, ul, li, form, hr, div, ...

  • none

    display none bisa digunakan untuk menghilangkan elemen tersebut

Sifat Value Display

1. inline
  • Elemen HTML yang secara default tidak menambahkan baris baru ketika dibuat
  • Lebar dan tinggi elemennya sebesar konten yang ada di dalamnya
  • Kita tidak dapat mengatur tinggi dan lebar dari elemen inline
  • Margin dan padding hanya mempengaruhi elemen secara horizontal, tidak vertikal
2. inline-block
  • Tidak ada elemen yang secara default memiliki properti display: inline-block;
  • Kita harus ubah secara manual properti tersebut
  • Perilaku dasarnya sama dengan elemen inline
  • Perbedaanya, elemen inline-block dapat kita atur tinggi dan lebarnya
3. block
  • Elemen HTML yang secara default menambahkan baris baru ketika dibuat
  • Jika tidak diatur lebarnya, maka lebar default dari elemen block akan memenuhi lebar dari browser / parentnya
  • Kita dapat mengatur tinggi dan lebar dari elemen block
  • Di dalam elemen block, kita dapat menyimpan tag dengan elemen inline, inline-block, atau bahkan elemen block lagi
4. none
  • Digunakan untuk menghilangkan elemen tersebut
  • Inilah yang sering saya gunakan untuk membuat website ini

Definisi

  • px

CSS Layouting Box Model

Definisi

CSS box model mendefinisikan 'kotak' yang dihasilkan oleh sebuah elemen, lalu menampilkannya sesuai dengan format visualnya. Berikut konsep pada box model

  • Setiap elemen di halaman berada di dalam sebuah box (kotak)
  • Kita bisa mengatur ukuran dan posisi kotak tersebut. Hanya elemen dengan display block
  • Kita bisa memberi warna / gambar sebagai background kotak tersebut

CSS Box Model

CSS box model terdiri dari 4 komponen yaitu margin, border, padding, dan content

  • margin

    area transparan di sekitar kotak (di luar border)

  • padding

    area transparan di dalam kotak (antara content dan border)

Margin

Sifat Properti
  • Margin adalah area transparan di sekitar kotak (di luar border)
  • Margin tidak bisa saling menambahkan secara vertikal (overlapping) tapi bisa saling menambahkan secara horizontal
  • Margin bisa diberi nilai negatif. Negatif margin akan membuat kotaknya bergerak berlawanan arah, dan bisa digunakan untuk menyembunyikan elemen
  • Margin bisa diberi nilai auto, khusus untuk margin kiri dan margin kanan. Nilai auto membuat elemen berada di tengah-tengah elemen parentnya
Shorthand

Shorthand digunakan untuk menyingkat penulisan. Shorthand untuk properti margin bisa memiliki 1 nilai, 2 nilai, 3 nilai, atau 4 nilai.

        
    selector {
        margin: top right bottom left;
    }
  • Jika 1 nilai maka nilai tersebut adalah margin dari setiap sisi kotak (elemen)
  • Jika 2 nilai maka nilai pertama adalah sisi atas dan bawah, dan nilai kedua adalah sisi kiri dan kanan, nilai kedua juga bisa gunakan nilai auto
  • Jika 3 nilai maka nilai pertama adalah sisi atas, nilai kedua adalah sisi kiri dan kanan (auto), dan nilai ketiga adalah sisi bawah
  • Jika 4 nilai maka keempat nilai tersebut adalah dimulai dari sisi atas, kanan, bawah, kiri (searah jarum jam)

Padding

Sifat Properti
  • Padding adalah area transparan di dalam kotak (antara content dan border)
  • Hampir sama dengan margin, bedanya adalah margin memberi jarak diluar elemen, sedangkan padding memberi jarak didalam elemen, jarak antara ujung elemen dengan kontenya
  • Padding tidak bisa diberi nilai negatif
  • Padding tidak punya nilai auto
  • Padding mempengaruhi ukuran dari kotak
Shorthand

Shorthand digunakan untuk menyingkat penulisan. Shorthand untuk properti padding bisa memiliki 1 nilai, 2 nilai, 3 nilai, atau 4 nilai.

        
    selector {
        padding: top right bottom left;
    }
  • Jika 1 nilai maka nilai tersebut adalah padding dari setiap sisi kotak (elemen)
  • Jika 2 nilai maka nilai pertama adalah sisi atas dan bawah, dan nilai kedua adalah sisi kiri dan kanan, nilai kedua tidak bisa gunakan nilai auto
  • Jika 3 nilai maka nilai pertama adalah sisi atas, nilai kedua adalah sisi kiri dan kanan (tidak bisa auto), dan nilai ketiga adalah sisi bawah
  • Jika 4 nilai maka keempat nilai tersebut adalah dimulai dari sisi atas, kanan, bawah, kiri (searah jarum jam)

Border

Sifat Properti
  • Border adalah batas di sekeliling content dan padding
  • Penulisan valuenya ada 3 bagian, pertama ketebalan, kedua jenis border (style), dan ketiga warna border
  • Ketebalan border bisa gunakan ukuran px (pixel)
  • Jenis border (style) ada 4
    • solid
    • dotted
    • dashed
    • double
  • Warna border bisa gunakan nama warna, hexadecimal, atau rgb
  • Border mempengaruhi ukuran dari kotak
Penulisan border

Penulisan border adalah sebagai berikut

        
    border: width style color;

Kita bisa memberikan border dengan sisi, ketebalan, style, dan warna yang kita tentukan sendiri

        
    selector {
        border-right: 2px solid red;
        border-bottom: 3px solid blue;
    }

Atau kita bisa berikan border pada setiap sisinya

        
    selector {
        border: 1px solid grey;
    }

Properti Khusus

Di dalam CSS Box Model terdapat properti khusus yang bisa digunakan untuk membuat ukuran dari kotak (elemen) tidak dapat dipengaruhi oleh padding atau border, properti itu adalah box-sizing.

        
    selector {
        box-sizing: border-box;
    }

Properti box-sizing digunakan supaya ukuran kotak (elemen) tidak berubah-ubah jika ditambahkan padding atau border.

Definisi

Float adalah properti pada CSS yang digunakan untuk mengatur posisi sebuah elemen. Sebuah elemen dapat dipaksa untuk berada di sebelah kiri atau kanan dari parent / pembungkusnya dengan menggunakan properti ini.

Float

  • right

  • normal flow

  • out-of flow

CSS Layouting Position

Definisi

CSS position ini sangat penting karena akan membuat kita dapat melakukan tata letak dari elemen-elemen HTML yang kita buat sesuai dengan keinginan kita.

Position

Position memungkinkan kita untuk menggeserkan sebuah elemen tanpa mempengaruhi elemen di sekitarnya. Untuk menggunakan CSS position kita butuh sebuah properti position dan value yang bisa kita gunakan ada 4, yaitu

  • static

    value default

  • relative

    membuat elemen bergerak relatif terhadap posisi semula-nya

  • absolute

    membuat elemen bergerak relatif terhadap posisi elemen parentnya, selama elemen parentnya memiliki posisi yang juga non-static

  • fixed

    elemen tersebut akan bergerak relatif terhadap window browsernya, dia tidak akan mempedulikan elemen parentnya meskipun parentnya non-static

Implementasi

1. static
  • static adalah nilai default dari setiap elemen ketika tidak diberi position
  • Menggunakan position selain static (non-static), akan membuat sebuah elemen seolah-olah berbeda dimensi dari elemen lainya
  • Elemen yang diberi position selain static dapat menggunakan properti top, left, bottom, dan right untuk mengatur posisinya
2. relative
  • Ketika kita menggerakan elemen dengan posisi relative (menggunakan properti top, left, bottom, dan right), ruang yang ditempati oleh elemen tersebut sebelumnya masih ada
  • Ketika kita menggerakan elemen dengan posisi relative, elemen akan bergerak relatif terhadap posisi semula-nya
  • Jika kita memberi properti top: 0; dan left: 0; maka elemen tidak berubah posisinya
3. absolute
  • Ketika kita menggerakan elemen dengan posisi absolute (menggunakan properti top, left, bottom, dan right), ruang yang ditempati oleh elemen tersebut sebelumnya dianggap tidak ada
  • Ketika kita menggerakan elemen dengan posisi absolute, elemen akan bergerak relatif terhadap posisi elemen parentnya, selama elemen parentnya memiliki posisi yang juga non-static
  • Jika kita memberi properti top: 0; dan left: 0; maka elemen akan berada di ujung kiri atas dari elemen parentnya
4. fixed
  • Ketika kita menggerakan elemen dengan posisi fixed (menggunakan properti top, left, bottom, dan right), maka dia akan bergerak relatif terhadap window browsernya, dia tidak akan mempedulikan elemen parentnya meskipun parentnya non-static
  • Ketika kita scroll halamanya, elemen tersebut akan terkunci di posisi itu
  • Posisi fixed biasa digunakan untuk membuat top bar

Z-index

Properti z-index digunakan untuk mengatur tumpukan dari elemen-elemen non-static relative, absolute, dan fixed. Caranya adalah dengan memberikan elemen tersebut properti z-index. Valuenya adalah angka, semakin besar nilai valuenya maka elemen tersebut akan berada di tumpukan paling depan.

        
    selector {
        position: absolute;
        z-index: 1;
    }

Apa itu FlexBox

Definisi

  • FlexBox atau Flexible Box Module adalah teknik pada CSS yang digunakan untuk membantu kita dalam mengatur layout atau posisi sebuah halaman website
  • FlexBox adalah model layout 1 dimensi yang dapat mengatur jarak dan penjajaran antar item dalam sebuah container
  • 1 Dimensi maksudnya adalah FlexBox hanya dapat mengatur 1 dimensi pada saat tertentu, antara baris atau kolom, tidak bisa keduanya sekaligus
  • Jika ingin bisa mengatur 2 dimensi sekaligus maka bisa gunakan CSS Grid

Istilah Pada Flexible Box Module

  • container

    elemen pembungkus dari elemen di dalamnya atau disebut (parent)

  • items

    semua elemen di dalam elemen pembungkus atau disebut (child)

  • main axis

    sumbu utama dari sebuah container yang menentukan urutan dari penempatan items secara horizontal

  • main-start | main-end

    mulai dan berakhirnya items yang disimpan di dalam container

  • main size

    ukuran (width/height) dari container yang akan membuat dimensi dari items nya relatif terhadap main size

Properti Pada Container

  • display

    berfungsi untuk membuat sebuah elemen parent menjadi flex box, dan membuat elemen-elemen di dalamnya bisa berperilaku flex juga

  • flex-direction

    berfungsi untuk mengatur arah dari elemen-elemen di dalamnya

  • flex-wrap

    secara default, semua items di dalam container akan berada pada satu baris meskipun ukurannya sudah tidak cukup, flex-wrap memungkinkan untuk memindahkan items ke baris di bawahnya

  • justify-content

    berfungsi untuk mengatur rata dari elemen-elemen di dalamnya

  • align-items

    berfungsi untuk mengatur kesejajaran antar item secara vertikal

  • align-content

    berfungsi untuk mengatur jarak antar items terhadap cross axis tapi hanya berlaku ketika content lebih dari satu baris

  • border-radius

  • border-bottom-right-radius

        height: 200px;

    }

  • px

  • %
    - relatif pada ukuran window browser

CSS 3 Gradients

Definisi

Gradients adalah teknik untuk memberikan warna gradasi pada elemen yang kita buat.

Gradients

Ada 2 tipe gradients yang bisa kita gunakan, ada linear-gradient dan radial-gradient.

  • linear-gradient

    memberikan gradasi warna secara garis lurus

  • radial-gradient

    memberikan gradasi warna secara melingkar

Implementasi

1. linear-gradient

linear-gradient memberikan gradasi warna secara garis lurus. linear-gradient adalah value dari properti background-image. value ini memiliki beberapa parameter.

Syntax sederhana
        
    background-image: linear-gradient(< warna-awal>, < warna-akhir>) ;
Syntax dengan sudut
        
    background-image: linear-gradient(< sudut> | < arah>, < warna-1>, < warna-2>, < warna-n>) ;
Syntax dengan color stop
        
    background-image: linear-gradient(< sudut> | < arah>, < warna-1>< color-stop>, < warna-2>< color-stop>, < warna-n>< color-stop>) ;
Penjelasan Value
  • < sudut>

    bisa diisi dengan numerik (deg) atau derajat

  • < arah>

    bisa diisi dengan to top/right/bottom/left atau diagonal to left top/left bottom/right top/right bottom

  • < warna>

    bisa diisi nama warna, hexadecimal, rgb atau rgba

  • < color-stop>

    nilai dalam (%) yang digunakan untuk memberikan batas warna dalam presentase yang di tentukan

2. radial-gradient

radial-gradient memberikan gradasi warna secara melingkar. radial-gradient adalah value dari properti background-image. value ini memiliki beberapa parameter.

Syntax sederhana
        
    background-image: radial-gradient(< warna-awal>, < warna-akhir>) ;
Syntax lengkap
        
    background-image: radial-gradient(
        < tipe> < jangkauan> at < posisi-y> < posisi-x>, 
        < warna-1>< color-stop>, 
        < warna-2>< color-stop>, 
        < warna-n>< color-stop>
    );
Penjelasan Value
  • < tipe>

    adalah tipe gradasinya. valuenya ada 2 ellipse & circle

  • < jangkauan>

    adalah jarak dari pusat gradient ke sisi terluarnya

    • closest-corner

      jarak dari pusat gradient ke sudut terdekat elemen

    • closest-side

      jarak dari pusat gradient ke sisi terdekat elemen

    • farthest-corner

      jarak dari pusat gradient ke sudut terjauh elemen

    • farthest-side

      jarak dari pusat gradient ke sisi terjauh elemen

  • < posisi-y>

    memindahkan pusat gradient secara horizontal

    • top
    • center
    • bottom
    • (px)

  • < posisi-x>

    memindahkan pusat gradient secara vertikal

    • left
    • center
    • right
    • (px)

  • < warna>

    bisa diisi nama warna, hexadecimal, rgb atau rgba

  • < color-stop>

    nilai dalam (%) yang digunakan untuk memberikan batas warna dalam presentase yang di tentukan

CSS 3 Filter

Definisi

Properti filter adalah properti yang digunakan untuk memberikan efek visual pada gambar, background, atau border.

Penulisan
filter: < tipe-filter> ;

Filter

Berikut beberapa tipe filter

  • blur(px)

    membuat gambar menjadi blur. semakin besar nilainya maka gambar akan semakin blur

  • brightness(angka | %)

    membuat gambar menjadi cerah. default 1 | 100%, jika nilainya ditambah maka gambar akan semakin cerah, dan jika nilainya dikurangi maka gambar akan semakin gelap

  • contrast(angka | %)

    sama dengan brightness

  • saturate(angka | %)

    membuat warna gambar menjadi semakin tajam. default 1 | 100%, jika nilainya ditambah maka warna akan semakin tajam, dan jika nilainya dikurangi maka warna akan semakin pudar

  • grayscale(angka | %)

    membuat warna gambar menjadi abu-abu. default 0 | 0%, kebalikan dari saturate

  • sepia(angka | %)

    sama dengan grayscale, bedanya dia tidak merubah menjadi abu-abu tapi merubah menjadi seperti warna kuning

  • hue-rotate(deg)

    merubah warna pada gambar dengan roda warna. nilainya 0-360 derajat

  • invert(angka | %)

    membuat warna gambar menjadi negatif. default 0 | 0%, jika nilainya ditambah maka warna akan semakin negatif

  • opacity(angka | %)

    membuat gambar menjadi transparan. default 1 | 100%, jika dikurangi maka gambar akan semakin transparan

  • drop-shadow(sama seperti box-shadow)

    untuk memberikan bayangan pada gambar maupun teks pada format gambar

CSS 3 Transform

Definisi

Properti transform memungkinkan kita dapat memanipulasi format visual dari elemen HTML. formatnya ada ukuran, bentuk, maupun posisi. properti transform ada 2 jenis yaitu 2D & 3D.

Transform

Fungsi pada transform

  • scale()

    memperbesar / memperkecil elemen

  • rotate()

    memutar elemen

  • skew()

    membuat elemen condong / miring

  • translate()

    menggeser posisi elemen

Implementasi

1. scale(angka)

Memperbesar atau memperkecil elemen.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid grey;
        transform: scale(2);
    }

Kotaknya akan membesar 2 kali lipat

Ada 3 jenis scale() berdasarkan sumbunya
  • scale(angka, angka)

    memperbesar elemen dengan sumbu X dan Y

  • scaleX(angka)

    memperbesar elemen secara horizontal

  • scaleY(angka)

    memperbesar elemen secara vertikal

2. rotate(deg)

Memutar elemen dari posisi semula, jika nilainya (+) dia akan berputar searah jarum jam, dan jika nilainya (-) dia akan berputar berlawanan arah jarum jam.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid grey;
        transform: rotate(45deg);
    }

Kotaknya akan berputar 45 derajat dari posisi semula

Ada 3 jenis rotate() berdasarkan sumbunya
  • rotate(deg)

    pusat putaran ada di tengah kotak

  • rotateX(deg)

    pusat putaran ada di sumbu horizontal, jadi dia akan berputar secara vertikal

  • rotateY(deg)

    pusat putaran ada di sumbu vertikal, jadi dia akan berputar secara horizontal

3. skew(deg)

Memiringkan elemen dari posisi semula, defaultnya dia akan memiringkan secara horizontal sama dengan skewX(deg).

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid grey;
        transform: skew(45deg);
    }

Kotaknya akan miring 45 derajat dari posisi semula secara horizontal

Ada 3 jenis skew() berdasarkan sumbunya
  • skew(deg)

    sama dengan skewX(deg)

  • skewX(deg)

    memiringkan secara horizontal

  • skewY(deg)

    memiringkan secara vertikal

4. translate(px | %)

Menggeser elemen dari posisi semula, jika nilainya (+) dia akan geser ke kanan, dan jika nilainya (-) dia akan geser ke kiri.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid grey;
        transform: translate(200px);
    }

Kotaknya akan geser ke kanan 200px dari posisi semula

Ada 3 jenis translate() berdasarkan sumbunya
  • translate(px, px)

    menggeser elemen dengan sumbu X dan Y

  • translateX(px)

    menggeser elemen secara horizontal

  • translateY(px)

    menggeser elemen secara vertikal

5. multiple value

Properti transform adalah salah satu properti yang valuenya bisa ditumpuk. bisa dengan jenis value yang sama.

transform: scaleX(1.5) scaleY(2);

Bahkan bisa dengan jenis value yang berbeda-beda

transform: scale(2) rotate(90deg) translate(200px);

CSS 3 Transition

Definisi

Properti transition memungkinkan kita dapat mengubah nilai dari properti HTML secara halus. di dalam CSS ada 2 properti yang digunakan untuk mengubah nilai properti HTML secara halus, ada transition dan animation, transition hanya memiliki 2 keadaan saja yaitu keadaan awal dan keadaan akhir, sedangkan animation bisa memiliki banyak keadaan.

Transition

Penulisan
transition: [property] < durasi> [fungsi] [delay] ;
Penjelasan Value
  • < durasi>

    valuenya second (s) atau milisecond (ms). memberi waktu perubahan dari keadaan awal dengan keadaan akhir

  • [property]

    menentukan properti apa saja yang akan dilakukan transisi

  • [fungsi]

    timing function atau pergerakan apa yang bisa dilakukan

  • [delay]

    valuenya second (s) atau milisecond (ms). memberikan waktu tunggu sebelum kita menjalankan transisi

Implementasi

1. < durasi>

Valuenya second (s) atau milisecond (ms). memberi waktu perubahan dari keadaan awal dengan keadaan akhir.

                                    
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid #00ff00;
        transition: 2s;
    }
    .container:hover .kotak {
        transform: scale(2);
    }
Jika kotak di sentuh maka kotak akan membesar 2 kali lipat dalam waktu 2 detik
2. [property]

Menentukan properti apa saja yang akan dilakukan transisi. bisa satu properti atau banyak properti.

                                    
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid #00ff00;
        transition: [border | transform | all | none] 2s;
    }
    .container:hover .kotak {
        transform: scale(2);
        border: 2px solid royalblue;
    }
  • Jika tidak dituliskan propertinya maka defaultnya adalah all
  • Jika dituliskan propertinya maka properti tersebut yang akan diberi transisi
  • Jika dituliskan none maka tidak ada transisi
3. [fungsi]

timing function atau pergerakan apa yang bisa dilakukan.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid #00ff00;
        transition: 2s ease;
    }
    .container:hover .kotak {
        transform: scale(2);
    }
Berikut beberapa contoh fungsi (timing function)
  • ease
  • ease-in
  • ease-out
  • ease-in-out
  • linear
  • cubic-bezier(w, x, y, z)
4. [delay]

Valuenya second (s) atau milisecond (ms). memberikan waktu tunggu sebelum kita menjalankan transisi.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid #00ff00;
        transition: 2s 1s;
    }
    .container:hover .kotak {
        transform: scale(2);
    }
Jika kotak di sentuh maka kotak akan membesar 2 kali lipat dalam waktu 2 detik setelah waktu tunggu 1 detik
5. multiple value

Properti transition bisa diisi lebih dari satu value.

        
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid #00ff00;
        transition: margin-left 3s, transform 3s 2s, background-color 500ms 3s;
    }
    .container:hover .kotak {
        transform: scale(2);
        margin-left: 200px;
        background-color: royalblue;
    }
Urutan transisi jika elemennya di sentuh
  • pertama akan geser 200px dalam 3 detik
  • kedua akan membesar 2 kali lipat setelah 2 detik
  • ketiga akan berubah warna royalblue dalam 500 milisecond setelah 3 detik

CSS 3 Animation

Definisi

Properti animation digunakan untuk membuat animasi dengan elemen HTML. berbeda dengan transition, animation bisa melakukan lebih dari dua keadaan (keyframes).

Keyframes

Berikut penulisan sederhana keyframes. Format ini bisa digunakan jika hanya ada 2 keadaan

                                    
    @keyframes [name] {
        from {
            [property-css]
        }
        to {
            [property-css]
        }
    }

Berikut penulisan keyframes menggunakan presentase. Format ini bisa digunakan jika punya banyak keadaan

                                    
    @keyframes [name] {
        0% {
            [property-css]
        }
        25% {
            [property-css]
        }
        50% {
            [property-css]
        }
        75% {
            [property-css]
        }
        100% {
            [property-css]
        }
    }

Animation

Cara menggunakannya adalah dengan memberikan properti animation pada selector yang ingin di animasi, dan tuliskan keyframes di luar selector.

                                    
    .kotak {
        width: 200px;
        height: 200px;
        border: 2px solid grey;
        animation: [name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill-mode] [play-state];
    }

    @keyframes [name] {
        from {
            [property-css]
        }
        to {
            [property-css]
        }
    }
Penjelasan Value
  • [name]

    nama dari animasi yang dibutuhkan pada keyframes nantinya

  • [duration]

    durasi dari animasinya

  • [timing-function]

    jenis pergerakan animasi. sama dengan transition

  • [delay]

    waktu tunggu sebelum animasi dilakukan

  • [iteration-count]

    jenis pengulangan animasi. valuenya

    • < angka>

      jumlah eksekusi animasi

    • keyword infinite

      animasi diulang terus-menerus

  • [direction]

    alur eksekusi animasi. valuenya

    • normal

      dari awal ke akhir

    • reverse

      dari akhir ke awal

    • alternate

      dari awal ke akhir berjalan 2 kali

    • alternate-reverse

      dari akhir ke awal berjalan 2 kali

  • [fill-mode]

    menentukan ending dari animasi. valuenya

    • none

      normal

    • forwards

      setelah animasi selesai, dia tidak kembali ke posisi awal

    • backwards

    • both

  • [play-state]

    valuenya

    • running

      berjalan normal

    • paused

      animasi berhenti. biasa digunakan dengan JavaScript

10 Prinsip Responsive Web Design

Definisi

Responsive Web Design adalah sebuah pendekatan yang menyarankan agar proses perancangan dan pembangunan sebuah web app harus merespon terhadap perilaku pengguna berdasarkan dari ukuran layar, platform, dan orientasi layar.

Prinsip Responsive Web Design

  • Adaptive vs Responsive Design

    satu design untuk berbagai ukuran layar

  • Viewport & Viewport Meta Tag

    tag untuk membuat web yang responsive

  • Breakpoints

    titik sebagai patokan CSS untuk berbagai ukuran layar

  • Media Query

    syntax untuk membuat responsive web design dengan Breakpoints

  • Fluid Layouts

    membuat elemen pada halaman layout dengan nilai yang relatif

  • Relative Units

    satuan relatif yang digunakan untuk membuat elemen yang relatif

  • Responsive Media

    membuat media seperti gambar dan video menjadi relatif

  • Flexbox

    display untuk mengatur elemen baris atau kolom

  • Grid

    display untuk mengatur elemen baris dan kolom

  • Mobile First Design

    prioritas desain yang sebaiknya pertama kali dibuat

Penjelasan

1. Adaptive vs Responsive Design
Adaptive
adalah teknik membuat banyak desain untuk banyak ukuran.
Responsive
adalah teknik membuat satu design untuk banyak ukuran sekaligus.
2. Viewport & Viewport Meta Tag

Viewport adalah halaman web yang terlihat oleh user. Viewport Meta Tag adalah tag yang digunakan untuk membuat halaman web responsive menyesuaikan ukuran lebar layar, tag ini di letakan di dalam tag head.

< meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0,user-scalable=no">
3. Breakpoints

Adalah titik-titik ukuran lebar layar yang biasanya digunakan sebagai patokan ketika kita ingin mengubah design kita.

Breakpoints Device
< 576px
 Extra Small Device / Portrait Phone
>= 576px
 Small Device / Landscape Phone
>= 768px
 Medium Device / Tablets
>= 992px
 Large Device / Laptop, Dekstop
>= 1200px
 Extra Large Device / Large Dekstop, Wide Screen
>= 1400px
 XX-Large Device
4. Media Query

Media Query adalah syntax CSS yang digunakan untuk membuat responsive web design dengan menggunakan Breakpoints.

                                        
    @media (min-width: 576px) { ... }

    @media (min-width: 768px) { ... }

    @media (min-width: 992px) { ... }

    @media (min-width: 1200px) { ... }

    @media (min-width: 1400px) { ... }
5. Fluid Layouts
Bagaimana cara kita untuk membuat layout yang fluid. caranya adalah dengan memberikan nilai dengan satuan yang relatif (%).
6. Relative Units

Satuan-satuan relatif yang akan membuat website kita menjadi fluid atau responsive.

Relative Units
  • %
  • rem & em
  • vw & vh
  • vmin & vmax
Ini adalah unit atau satuan yang termasuk ke dalam kategori relatif
7. Responsive Media
Bagaimana cara kita menyimpan atau menyisipkan media ke dalam website kita secara responsive. media tersebut antara lain adalah image dan video.
8. Flexbox
Display baru yang digunakan untuk mengatur beberapa elemen sekaligus yang terdapat dalam satu baris atau satu kolom.
9. Grid
Sama dengan Flexbox, kelebihanya adalah dia bisa mengatur banyak elemen sekaligus tidak peduli elemen berada di baris atau kolom yang berbeda.
10. Mobile First Design

Untuk membuat website yang responsive kita sebaiknya mendahulukan untuk membuat ukuran device mobile terlebih dahulu, setelah selesai kita bisa lanjut untuk ukuran yang lebih besar, alasanya antara lain

  • Pengguna mobile jauh lebih banyak
  • Ragam mobile device lebih banyak
  • 'Content First' / Hierarki Informasi
