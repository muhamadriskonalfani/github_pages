Pengenalan Class Dan Object

Definisi

1. Class

Class adalah kumpulan variabel, method, dan konstruktor yang digunakan untuk mendefinisikan objek yang akan dibuat

  • Atribut

    Atribut adalah variabel yang menyimpan data dalam objek

  • Method

    Method adalah fungsi atau prosedur yang melakukan tugas tertentu pada objek

  • Konstruktor

    Konstruktor adalah sebuah metode khusus dalam sebuah class yang digunakan untuk menginisialisasi objek dari class tersebut

        
    public class Hero {

        // variabel
        String name;
        double health;

        // konstruktor
        Player(String nameHero, double healthHero){
            this.name = nameHero;
            this.health = healthHero;
        }

        // method
        void display(){
            System.out.println("Name : " + this.name);
            System.out.println("Health : " + this.health);
        }
    }
2. Object

Object, di sisi lain, adalah instansi atau realisasi dari class. Dengan menggunakan class sebagai blueprint, kita dapat membuat banyak objek dengan atribut dan metode yang sama tetapi nilai yang berbeda

        
    public class Main {
        public static void main(String[] args) {
            
            // membuat object
            Hero hero1 = new Hero("ucup", 100);
            Hero hero2 = new Hero("otong", 80);

            // menampilkan object dengan method 
            hero1.display();
            hero2.display();
        }
    }

Mendefinisikan object

1. Cara Biasa

Cara ini dilakukan dengan kita membuat object dan mendefinisikan object secara terpisah

        
    public class Hero {

        // variabel
        String name;
        double health;
    }

    public class Main {
        public static void main(String[] args) {
            
            // membuat object
            Hero hero1 = new Hero();
            Hero hero2 = new Hero();

            // mendefinisikan object
            hero1.name = "ucup";
            hero1.health = 100;

            hero2.name = "otong";
            hero2.health = 80;
        }
    }
2. Konstruktor

Konstruktor memungkinkan kita untuk mendefinisikan object saat object dibuat. Konstruktor memudahkan kita untuk mendefinisikan object yang banyak dengan singkat

        
    public class Hero {

        // variabel 
        String name;
        double health;

        // konstruktor
        Hero(String nameHero, double healthHero){
            this.name = nameHero;
            this.health = healthHero;
        }
    }

    public class Main {
        public static void main(String[] args) {
            
            // object dengan konstructor
            Hero hero1 = new Hero("ucup", 100);
            Hero hero2 = new Hero("otong", 80);
        }
    }

Menampilkan object

1. Cara Biasa

Cara ini dilakukan dengan menuliskan perintah System.out.println(...) dan diisi dengan object yang ingin ditampilkan

        String name;
        double health;

        }

Cara ini memudahkan kita untuk manampilkan object, cara ini dilakukan dengan membuat method di dalam class, dan dipanggil setelah object

            Hero hero2 = new Hero("otong", 80);

            hero2.display();

Constructor

Definisi

Konstruktor adalah suatu fungsi atau method yang pertama kali dipanggil saat object dibuat. Konstruktor adalah metode khusus yang digunakan untuk menginisialisasi objek yang baru dibuat. Konstruktor memiliki nama yang sama dengan nama kelasnya dan tidak mengembalikan nilai apapun.

Karakteristik

Konstruktor pada Java memiliki beberapa karakteristik yang penting

  • Konstruktor harus memiliki nama yang sama dengan nama kelas tempat ia didefinisikan.
  • Konstruktor tidak memiliki tipe pengembalian, termasuk void. Sebagai gantinya, mereka digunakan untuk menginisialisasi objek.
  • Konstruktor dipanggil secara otomatis saat objek baru dibuat dengan menggunakan operator "new".
  • Konstruktor juga dapat memiliki parameter untuk menerima nilai awal yang akan digunakan dalam proses inisialisasi objek.

Method

Definisi

Method adalah sebuah fungsi yang bisa diberikan kepada objek, dan nantinya objek tersebut akan melakukan perintah yang ada di dalam fungsi tersebut.

Membuat Method

Untuk membuat metode dalam Java, Anda perlu mendeklarasikannya di dalam kelas menggunakan sintaks berikut 

                                        
    modifier return_type namaMethod(parameter) {
        // Blok kode
    }
  • Modifier mengontrol aksesibilitas dan sifat lainnya dari metode (misalnya, public, private, protected).
  • Return type adalah tipe data dari nilai yang dikembalikan oleh metode. Jika metode tidak mengembalikan nilai, gunakan kata kunci void.
  • Nama metode adalah nama yang digunakan untuk memanggil metode.
  • Parameter (opsional) adalah nilai yang diterima oleh metode untuk digunakan di dalamnya.
                                        
    public void display() {
        System.out.println("Name : " + this.name);
        System.out.println("Health : " + this.health);
    }

Memanggil Method

Untuk memanggil method di java, masuk kedalam class Main, ketik nama objek. diikuti dengan nama method dan parameter jika ada. contoh 

                                        
    objek.method(); // memanggil method tanpa parameter
    objek.method(nilai); // memanggil method dengan parameter

Pengembalian Nilai

Jika method mengembalikan nilai, anda perlu menggunakan kata kunci return di dalam blok kode untuk mengembalikan nilai tersebut 

                                        
    public int tambah(int a, int b) {
        return a + b;
    }
anda dapat menggunakan nilai yang dikembalikan dari method ini dalam bagian lain dari program anda.

Jenis Method

Method tanpa return dan tanpa parameter
        
    void show() {
        System.out.println("Nama : " + this.nama);
        System.out.println("NIM : " + this.NIM);
    }
Method tanpa return dan dengan parameter
        
    void setNama(String nama) {
        this.nama = nama;
    }
    void setNIM(String NIM) {
        this.NIM = NIM;
    }
Method dengan return dan tanpa parameter
        
    String getName() {
        return this.name;
    }
    String getNIM() {
        return this.NIM;
    }
Method dengan return dan dengan parameter
        
    String sayHi(String pesan) {
        return pesan + ", perkenalkan nama saya " + this.nama;
    }

Encapsulation

Definisi

Encapsulation adalah salah satu konsep dalam pemrograman berorientasi objek yang menggabungkan data (atribut) dan metode (fungsi) yang berhubungan dalam sebuah kelas, dan membatasi akses langsung ke data tersebut dari luar kelas. Hal ini dilakukan dengan menggunakan pengaturan aksesibilitas yang ditentukan dengan kata kunci seperti private, public, atau protected.

Kata Kunci

1. private
Ketika suatu anggota (atribut atau metode) dinyatakan sebagai "private", maka anggota tersebut hanya dapat diakses atau dimodifikasi oleh kode di dalam kelas itu sendiri, dan tidak dapat diakses oleh kode di kelas lain.
Berikut fungsi keyword private
  • Menyembunyikan implementasi internal suatu kelas dari pengguna eksternal.
  • Mencegah perubahan yang tidak terkontrol.
  • Membatasi akses langsung ke data dalam sebuah objek.
  • Membantu dalam menjaga integritas data dengan membatasi akses langsung ke atribut.
        
    class Player {
        private String name;
        private double health;

        Player(String name, double health) {
            this.name = name;
            this.health = health;
        }
    }
variabel name dan health tidak bisa di akses class lain
2. public
Ketika suatu anggota (atribut atau metode) dinyatakan sebagai "public", maka anggota tersebut dapat diakses dan digunakan oleh kode di dalam kelas itu sendiri maupun oleh kode di kelas lain.
Berikut fungsi keyword public
  • Memungkinkan suatu anggota (atribut atau metode) dapat diakses dari dalam kelas itu sendiri maupun dari kelas lain.
  • Memungkinkan akses langsung ke atribut dari luar kelas, sehingga atribut tersebut dapat dibaca atau dimodifikasi langsung oleh kode di kelas lain.
  • Memungkinkan kelas lain untuk menggunakan kelas yang dideklarasikan sebagai "public" sebagai tipe data, objek, atau sebagai dasar untuk pewarisan.
        
    public class Car {
        public String brand;
        public String model;

        public void startEngine() {
            // Implementasi untuk memulai mesin mobil
            // ...
        }
    }

    public class Main {
        public static void main(String[] args) {
            Car myCar = new Car();
            myCar.brand = "Toyota";
            myCar.model = "Camry";
            myCar.startEngine();
        }
    }
3. protected
Ketika suatu anggota dinyatakan sebagai protected, anggota tersebut dapat diakses oleh kode di dalam kelas itu sendiri, oleh kelas-kelas turunan (subclass), dan oleh kelas-kelas dalam paket yang sama.
Berikut fungsi keyword protected
  • Memungkinkan anggota suatu kelas dapat diakses oleh kelas turunan (subclass).
  • Memungkinkan anggota suatu kelas dapat diakses oleh kelas-kelas dalam paket yang sama.
        
    public class Animal {
        protected String name;

        protected void sleep() {
            System.out.println("Animal is sleeping");
        }
    }

    public class Cat extends Animal {
        public void play() {
            System.out.println("Cat is playing");
        }

        public void printName() {
            System.out.println("Name: " + name);
        }
    }

Method Getter Dan Setter

Definisi

Getter dan setter adalah metode atau fungsi yang digunakan untuk mengakses (getter) dan memperbarui (setter) nilai properti (variabel) dalam suatu objek.

Getter
Getter adalah metode yang digunakan untuk mengakses (membaca) nilai properti dari suatu objek. Biasanya, nama getter dimulai dengan prefiks "get" diikuti dengan nama properti yang ingin diakses. Getter tidak menerima argumen dan mengembalikan nilai dari properti yang diakses.
Setter
Setter adalah metode yang digunakan untuk memperbarui (menulis) nilai properti suatu objek. Nama setter biasanya dimulai dengan prefiks "set" diikuti dengan nama properti yang ingin diubah. Setter menerima argumen yang merupakan nilai baru yang akan diberikan kepada properti.

Implementasi

class Person
                
    public class Person {
        private String name;
        private int age;

        public String getName() {
            return name;
        }

        public void setName(String name) {
            this.name = name;
        }

        public int getAge() {
            return age;
        }

        public void setAge(int age) {
            if (age >= 0) {
                this.age = age;
            } else {
                System.out.println("Usia tidak valid.");
            }
        }
    }
class Main
                
    public class Main {
        public static void main(String[] args) {
            Person person = new Person();
            person.setName("John");
            person.setAge(25);

            System.out.println("Name: " + person.getName());
            System.out.println("Age: " + person.getAge());
        }
    }
tampilan
                
    Name: John
    Age: 25

Static Keyword

Definisi

Kata kunci "static" digunakan untuk menyatakan bahwa suatu anggota kelas (baik metode atau variabel) terkait dengan kelas itu sendiri, bukan dengan instance atau objek kelas tersebut. Artinya, anggota "static" dapat diakses langsung melalui kelas tanpa harus membuat instance atau objek dari kelas tersebut.

Fungsi Static Keyword

Penggunaan kata kunci "static" memiliki beberapa fungsi utama

  • Metode Static

    Ketika sebuah metode dideklarasikan sebagai "static", maka metode tersebut dapat dipanggil langsung melalui kelas tanpa harus membuat objek dari kelas tersebut.

  • Variabel Static

    Ketika sebuah variabel dideklarasikan sebagai "static", maka variabel tersebut menjadi milik kelas, bukan milik setiap objek yang dibuat dari kelas tersebut.

Implementasi

Berikut adalah contoh penggunaan kata kunci "static" dalam Java OOP

class ContohKelas
                
    public class ContohKelas {
        // Variabel static
        public static int jumlahObjek = 0;

        // Metode static
        public static void cetakJumlahObjek() {
            System.out.println("Jumlah objek: " + jumlahObjek);
        }

        public ContohKelas() {
            // Menginkrementasi jumlah objek setiap kali objek baru dibuat
            jumlahObjek++;
        }

        public static void main(String[] args) {
            // Memanggil metode static tanpa membuat objek
            ContohKelas.cetakJumlahObjek();

            // Membuat beberapa objek
            ContohKelas objek1 = new ContohKelas();
            ContohKelas objek2 = new ContohKelas();

            // Memanggil metode static setelah objek dibuat
            ContohKelas.cetakJumlahObjek();
        }
    }
Penjelasan
  • Dalam contoh di atas, kelas ContohKelas yang memiliki variabel static jumlahObjek untuk menghitung jumlah objek yang telah dibuat.
  • Metode cetakJumlahObjek() adalah metode static yang mencetak nilai variabel static tersebut.
  • Dalam metode main(), kita dapat memanggil metode cetakJumlahObjek() langsung melalui kelas tanpa harus membuat objek terlebih dahulu. Setiap kali objek ContohKelas dibuat, nilai jumlahObjek akan ditingkatkan, dan hasilnya akan dicetak menggunakan metode static cetakJumlahObjek().
tampilan
                
    Jumlah objek: 0
    Jumlah objek: 2
Penjelasan
  • Pada baris pertama, metode cetakJumlahObjek() dipanggil tanpa membuat objek terlebih dahulu. Karena belum ada objek yang dibuat, nilai jumlahObjek masih 0, dan hasilnya dicetak sebagai "Jumlah objek: 0".
  • Kemudian, pada baris berikutnya, dua objek ContohKelas dibuat dengan menggunakan sintaks new ContohKelas(). Setiap kali objek dibuat, konstruktor akan dipanggil, dan dalam konstruktor, nilai jumlahObjek akan ditingkatkan.
  • Setelah objek-objek dibuat, metode cetakJumlahObjek() dipanggil lagi. Kali ini, karena sudah ada dua objek yang dibuat sebelumnya, nilai jumlahObjek menjadi 2, dan hasilnya dicetak sebagai "Jumlah objek: 2".

Inheritance

Definisi

Inheritance memungkinkan Anda untuk membuat kelas baru berdasarkan kelas yang ada. Kelas baru ini akan mewarisi atribut dan metode dari kelas yang ada, sehingga memungkinkan untuk membangun hierarki kelas. Dalam Java, pewarisan dilakukan dengan menggunakan kata kunci extends.

Implementasi

Berikut contoh penerapan inheritance pada java OOP 

            
    package Latihan_18;

    // inheritance adalah hubungan is_a

    // super class, parent class, base class
    class Hero{
        String name;
        
        // method
        void display(){
            System.out.println("Name : " + this.name);
        }
    }

    // subclass, child class, derived class
    class HeroStrength extends Hero{
        // kosong
    }

    class HeroIntelligent extends Hero{
        // kosong
    }

    public class Main {
        public static void main(String[] args){
            Hero hero1 = new Hero();
            hero1.name = "ucup";
            hero1.display();
            
            HeroStrength hero2 = new HeroStrength();
            hero2.name = "otong";
            hero2.display();
            
            HeroIntelligent hero3 = new HeroIntelligent();
            hero3.name = "Michael";
            hero3.display();
        }   
    }
Penjelasan
  • class Hero adalah kelas yang mewarisi, bisa disebut juga sebagai super class, parent class, base class
  • class HeroStrength dan HeroIntelligent adalah kelas yang diwarisi, bisa juga disebut sebagai subclass, child class, derived class
  • dengan keyword extends diikuti dengan nama class (parent class), maka sebuah class akan memiliki atribut (variabel dan method) dari parent class
  • dengan begitu, object yang dibuat dari class HeroStrength dan HeroIntelligent, yaitu hero2 dan hero3 dia juga memiliki variabel String name dan method display()

Overloading Methods

Definisi

Overloading pada Java OOP adalah konsep di mana sebuah kelas memiliki beberapa metode dengan nama yang sama tetapi memiliki parameter yang berbeda. Dalam overloading, nama metode tetap sama, tetapi tipe, urutan, atau jumlah parameter berbeda. Ini memungkinkan pemanggilan metode dengan nama yang sama tetapi dengan argumen yang berbeda untuk menghasilkan perilaku yang berbeda.

Java memungkinkan overloading metode untuk memberikan fleksibilitas dalam pemrograman, karena Anda dapat menggunakan nama metode yang sama untuk melakukan tugas yang berbeda tergantung pada argumen yang diberikan. Misalnya, Anda dapat memiliki metode "hitungLuas()" yang menghitung luas persegi dan metode "hitungLuas()" yang menghitung luas lingkaran.

Implementasi

Contoh sederhana overloading metode dalam Java

class Calculator
        
    public class Calculator {
        public int add(int a, int b) {
            return a + b;
        }

        public double add(double a, double b) {
            return a + b;
        }

        public int add(int a, int b, int c) {
            return a + b + c;
        }
    }
Penjelasan
  • Dalam contoh di atas, kelas Calculator memiliki tiga metode add() yang merupakan contoh overloading.
  • Metode pertama menerima dua parameter integer dan mengembalikan hasil penjumlahan integer.
  • Metode kedua menerima dua parameter double dan mengembalikan hasil penjumlahan double.
  • Metode ketiga menerima tiga parameter integer dan mengembalikan hasil penjumlahan ketiga parameter tersebut.

Anda dapat memanggil metode-metode ini dengan cara yang berbeda tergantung pada jenis dan jumlah argumen yang Anda berikan

class Main
        
    Calculator calculator = new Calculator();
    int result1 = calculator.add(2, 3);          // Memanggil add(int, int)
    double result2 = calculator.add(2.5, 3.7);   // Memanggil add(double, double)
    int result3 = calculator.add(2, 3, 4);       // Memanggil add(int, int, int)
Dalam hal ini, metode yang sesuai akan dipilih berdasarkan jenis dan jumlah argumen yang Anda berikan saat memanggil metode. Ini memberikan kemampuan yang lebih besar untuk menyesuaikan fungsionalitas metode sesuai dengan kebutuhan Anda.

Overriding Methods

Definisi

Overriding adalah konsep di mana sebuah subclass mengganti implementasi dari metode yang telah didefinisikan dalam superclassnya. Dengan overriding, subclass memiliki kemampuan untuk memberikan implementasi khusus untuk metode yang sama yang telah didefinisikan dalam superclass.

Untuk melakukan overriding dalam Java, metode di subclass harus memiliki nama, tipe pengembalian (return type), dan parameter yang sama dengan metode yang akan di-override di superclass. Kata kunci @Override dapat digunakan di atas metode di subclass untuk memberi tahu kompiler bahwa kita bermaksud untuk melakukan overriding.

Implementasi

Berikut adalah contoh penggunaan overriding dalam Java OOP 

                
    class Animal {
        public void makeSound() {
            System.out.println("The animal makes a sound.");
        }
    }

    class Cat extends Animal {
        @Override
        public void makeSound() {
            System.out.println("Meow!");
        }
    }

    public class Main {
        public static void main(String[] args) {
            Animal animal = new Animal();
            animal.makeSound();

            Cat cat = new Cat();
            cat.makeSound();
        }
    }
Penjelasan
  • Pada contoh di atas, kelas Cat merupakan subclass dari Animal.
  • Metode makeSound() pada kelas Animal di-override di kelas Cat untuk memberikan implementasi yang berbeda.
  • Ketika metode makeSound() dipanggil pada objek Animal, outputnya adalah "The animal makes a sound."
  • Sedangkan ketika metode tersebut dipanggil pada objek Cat, outputnya adalah "Meow!".

Subclass Constructor

Definisi

Subclass konstruktor adalah konstruktor yang didefinisikan di dalam kelas anak (subclass) yang diwariskan dari kelas induk (superclass). Konstruktor digunakan untuk membuat objek dari kelas tersebut. Berikut cara membuat subclass konstruktor

  • Ketik nama konstruktor sesuai nama subclassnya
  • Beri parameter sama dengan superclassnya, boleh ditambahkan parameter baru
  • parameter yang sama dengan yang ada di superclass harus dimasukan kedalam keyword super()
  • parameter yang baru dibuat di assign ke variabel global subclass tersebut

Polymorphism

Definisi

Polymorphism memungkinkan objek untuk memiliki banyak bentuk. Dalam Java, ini dapat dicapai melalui penggunaan konsep overloading dan overriding. Overloading adalah membuat metode dengan nama yang sama tetapi dengan parameter yang berbeda, sedangkan overriding adalah mengimplementasikan kembali metode yang telah didefinisikan dalam kelas induk.

Polymorphism adalah konsep di mana suatu objek dapat merujuk pada objek dari berbagai tipe yang berbeda dan masih dapat berperilaku sesuai dengan jenis objek yang dirujuk. Dalam konteks Java, terdapat dua jenis polimorfisme: polimorfisme statis (compile-time) dan polimorfisme dinamis (runtime).

Implementasi

Berikut adalah contoh sederhana polymorphism 

            
    class Hewan {
        public void suara() {
            System.out.println("Hewan membuat suara...");
        }
    }

    class Kucing extends Hewan {
        @Override
        public void suara() {
            System.out.println("Meong!");
        }
    }

    class Anjing extends Hewan {
        @Override
        public void suara() {
            System.out.println("Guk guk!");
        }
    }

    public class Main {
        public static void main(String[] args) {
            Hewan hewan1 = new Kucing();
            Hewan hewan2 = new Anjing();
            
            hewan1.suara(); // Output: Meong!
            hewan2.suara(); // Output: Guk guk!
        }
    }
Penjelasan
  • Pada contoh di atas, terdapat tiga kelas: Hewan, Kucing, dan Anjing.
  • Ketiga kelas ini memiliki metode suara() yang di-override.
  • Dalam metode main(), objek hewan1 dan hewan2 dideklarasikan sebagai objek tipe Hewan, tetapi merujuk pada objek Kucing dan Anjing secara berturut-turut.
  • Ketika metode suara() dipanggil pada kedua objek, JVM (Java Virtual Machine) akan memilih implementasi yang sesuai berdasarkan jenis objek yang dirujuk (polimorfisme dinamis).

Abstraksi

Definisi

Abstract class (kelas abstrak) adalah sebuah kelas yang tidak dapat diinstansiasi (tidak dapat membuat objek langsung dari kelas tersebut). Abstract class digunakan sebagai kerangka dasar untuk kelas-kelas turunan (subclasses) yang lebih spesifik.

Sebuah abstract class mungkin berisi implementasi (kode) dan/atau metode-metode abstrak. Metode abstrak adalah metode yang dideklarasikan tanpa memiliki implementasi di dalam abstract class. Kelas turunan yang mewarisi dari abstract class harus memberikan implementasi untuk semua metode abstrak yang dideklarasikan dalam abstract class tersebut.

Implementasi

Abstract class digunakan untuk memodelkan konsep umum yang bisa diwarisi oleh kelas-kelas turunannya. Misalnya, jika kita memiliki kelas "Hewan" sebagai abstract class, kita dapat mendefinisikan metode abstrak "makan" di dalamnya. Kemudian, kelas-kelas turunan seperti "Kucing" dan "Anjing" dapat mewarisi abstract class "Hewan" dan memberikan implementasi yang spesifik untuk metode "makan" sesuai dengan perilaku masing-masing hewan tersebut.

            
    abstract class Hewan {
        public abstract void makan();
    }

    class Kucing extends Hewan {
        public void makan() {
            System.out.println("Kucing sedang makan ikan.");
        }
    }

    class Anjing extends Hewan {
        public void makan() {
            System.out.println("Anjing sedang makan daging.");
        }
    }

    public class Main {
        public static void main(String[] args) {
            Hewan hewan1 = new Kucing();
            Hewan hewan2 = new Anjing();
            
            hewan1.makan(); // Output: Kucing sedang makan ikan.
            hewan2.makan(); // Output: Anjing sedang makan daging.
        }
    }
Penjelasan
  • Dalam contoh di atas, abstract class Hewan memiliki metode abstrak makan().
  • Kelas Kucing dan Anjing mewarisi Hewan dan memberikan implementasi khusus untuk metode makan().
  • Pada saat eksekusi, kita membuat objek "hewan1" dan "hewan2" dengan tipe data Hewan, namun menggunakan instansiasi kelas Kucing dan Anjing.
  • Ketika memanggil metode makan() pada objek-objek tersebut, akan mencetak hasil yang sesuai dengan implementasi yang diberikan di kelas-kelas turunan.

ArrayList Part 1

Definisi

ArrayList dalam Java adalah kelas yang terdapat dalam library java.util yang mengimplementasikan antarmuka List. ArrayList adalah struktur data dinamis yang berbasis pada array yang dapat digunakan untuk menyimpan dan mengelola sekumpulan elemen objek.

Dalam paradigma pemrograman berorientasi objek (OOP), ArrayList adalah sebuah kelas yang memungkinkan kita untuk membuat objek ArrayList yang dapat menyimpan objek-objek dengan tipe data yang sama atau berbeda. ArrayList memungkinkan kita untuk menambahkan, mengakses, mengubah, dan menghapus elemen-elemen dengan mudah.

Dalam ArrayList, elemen-elemennya diindeks secara berurutan dimulai dari indeks 0. ArrayList secara otomatis mengelola ukuran internalnya sesuai dengan jumlah elemen yang ditambahkan atau dihapus, sehingga tidak perlu khawatir tentang ukuran array yang tetap seperti dalam array biasa.

Dengan menggunakan ArrayList, kita dapat melakukan berbagai operasi seperti menambahkan elemen, mengakses elemen, mengubah elemen, menghapus elemen, mencari elemen, menghitung ukuran ArrayList, mengurutkan elemen, dan banyak lagi. ArrayList juga menyediakan metode dan fitur lain seperti pengulangan (iterasi) elemen dengan loop for-each, mengkonversi ArrayList menjadi array, dan menggabungkan dua ArrayList.

ArrayList Part 2

Fungsi Pada ArrayList

Dengan menggunakan ArrayList dalam Java, Anda dapat melakukan berbagai operasi dan manipulasi data, termasuk

1. Menambahkan elemen

Anda dapat menambahkan elemen baru ke ArrayList menggunakan metode add() atau addAll().

        
    import java.util.ArrayList;

    public class AddElementExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();

            // Menambahkan elemen baru
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            System.out.println(myArrayList);
        }
    }
2. Mengakses elemen

Anda dapat mengakses elemen dalam ArrayList menggunakan indeks dengan menggunakan metode get().

        
    import java.util.ArrayList;

    public class AccessElementExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            // Mengakses elemen menggunakan indeks
            String element = myArrayList.get(1);
            System.out.println("Elemen: " + element);
        }
    }
3. Mengubah elemen

Anda dapat mengubah nilai elemen yang ada dalam ArrayList dengan menggunakan metode set().

        
    import java.util.ArrayList;

    public class UpdateElementExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            // Mengubah elemen pada indeks tertentu
            myArrayList.set(1, "Pisang");

            System.out.println(myArrayList);
        }
    }
4. Menghapus elemen

Anda dapat menghapus elemen dari ArrayList menggunakan metode remove() dengan indeks atau nilai tertentu.

        
    import java.util.ArrayList;

    public class RemoveElementExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            // Menghapus elemen menggunakan indeks
            myArrayList.remove(1);

            System.out.println(myArrayList);
        }
    }
5. Mencari elemen

Anda dapat mencari elemen dalam ArrayList menggunakan metode indexOf() atau contains().

        
    import java.util.ArrayList;

    public class SearchElementExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            // Mencari indeks elemen menggunakan nilai
            int index = myArrayList.indexOf("Mangga");
            System.out.println("Indeks: " + index);

            // Mengecek apakah elemen ada dalam ArrayList
            boolean contains = myArrayList.contains("Jeruk");
            System.out.println("Ada dalam ArrayList? " + contains);
        }
    }
6. Menghitung ukuran ArrayList

Anda dapat mengetahui jumlah elemen dalam ArrayList menggunakan metode size().

        
    import java.util.ArrayList;

    public class ArrayListSizeExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            int size = myArrayList.size();
            System.out.println("Ukuran ArrayList: " + size);
        }
    }
7. Menggabungkan ArrayList

Anda dapat menggabungkan dua ArrayList menggunakan metode addAll().

        
    import java.util.ArrayList;
    import java.util.Arrays;

    public class MergeArrayListExample {
        public static void main(String[] args) {
            ArrayList< String> list1 = new ArrayList< String>(Arrays.asList("Apel", "Mangga"));
            ArrayList< String> list2 = new ArrayList< String>(Arrays.asList("Jeruk", "Anggur"));

            // Menggabungkan dua ArrayList
            list1.addAll(list2);

            System.out.println("Setelah menggabungkan:");
            System.out.println(list1);
        }
    }

12. Mengkonversi ArrayList menjadi array

Anda dapat mengonversi ArrayList menjadi array menggunakan metode toArray().

        
    import java.util.ArrayList;

    public class ArrayListToArrayExample {
        public static void main(String[] args) {
            ArrayList< String> myArrayList = new ArrayList< String>();
            myArrayList.add("Apel");
            myArrayList.add("Mangga");
            myArrayList.add("Jeruk");

            // Mengkonversi ArrayList menjadi array
            String[] array = myArrayList.toArray(new String[myArrayList.size()]);

            System.out.println("Array hasil konversi:");
            for (String element : array) {
                System.out.println(element);
            }
        }
    }
