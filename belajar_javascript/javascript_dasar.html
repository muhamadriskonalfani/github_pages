Operator Pada JavaScript

Definisi

Operator dalam javascript adalah sebuah simbol yang digunakan untuk melakukan operasi.

Jenis Operator

Ada banyak operator pada javascript, diantaranya

  • Aritmatika
  • Penugasan / Assignment
  • Perbandingan / Comparison
  • Logika
  • String
  • Kondisional
  • Typeof
Operator ini bisa dibagi berdasarkan jumlah operandnya, ada opertaor binary yaitu operator yang membutuhkan 2 operand, ada operator unary yaitu operator yang membutuhkan 1 operand, dan ada ternary yaitu operator yang membutuhkan 3 operand.

1. Aritmatika

Aritmatika adalah operator binary yang digunakan untuk melakukan operasi matematika, operator atirmatika antara lain ada tambah( + ), kurang( - ), kali( * ), bagi( / ), dan modulus atau sisa bagi( % ). Pada JavaScript terdapat urutan operasi aritmatika (operator precedence) yaitu kurung kali bagi tambah kurang atau bisa disingkat KuKaBaTaKu ( () * / + - ).

2. Penugasan / Assignment

Operator penugasan adalah operator binary. Yang paling sering digunakan pada operator ini adalah tanda sama dengan ( = ), dan ini biasanya digunakan saat kita bekerja dengan variabel.

Operator Penugasan sama saja dengan
x += y x = x + y
x -= y x = x - y
x *= y x = x * y
x /= y x = x / y
x %= y x = x % y

3. Perbandingan / Comparison

Operator perbandingan adalah operator binary yang digunakan untuk membandingkan 2 buah operand, operator perbandingan ini nantinya akan menghasilkan sebuah nilai boolean true atau false.

Operator Perbandingan Keterangan
== sama dengan
!= tidak sama dengan
=== strict sama dengan
!== strict tidak sama dengan
> lebih besar dari
< lebih kecil dari
>= lebih besar sama dengan
<= lebih kecil sama dengan
Perbedaan antara == dan === adalah jika == digunakan untuk membandingkan nilainya saja tidak dengan tipe datanya, sedangkan === digunakan untuk membandingkan nilai dan tipe datanya, contoh 10 == '10' hasilnya true dan jika 10 === '10' hasilnya false karena beda tipe datanya.

4. Logika

Operator logika adalah operator binary yang digunakan untuk menentukan logika dari beberapa ekspresi yang digabungkan.

Operator Logika Keterangan
&& and
|| or
! not

5. String

Operator string adalah operator binary. Di dalam javascript ini tanda + memiliki dua fungsi yaitu jika operandnya itu angka maka dia akan menjadi operator aritmatika, sedangakan jika operandnya adalah string maka dia akan menjadi operator penggabung string.

Contoh Hasil
10 + "10" "1010"
"10" + 10 "1010"
10 + 10 + "10" "2010"
"10" + 10 + 10 "101010"
Jika angka dengan angka maka hasilnya angka, sedangkan jika salah satunya string maka tanda ( + ) ini akan berubah menjadi operator string.

6. Typeof

Operator typeof adalah operator unary, operator ini digunakan untuk mengetahui tipe data apa dari sebuah nilai. cara penulisanya typof(operand). contoh typeof(10) hasilnya number, typeof("10") hasilnya string, dan typeof(true) hasilnya boolean.

7. Kondisional

Operator kodisional adalah operator ternary yang digunakan untuk melakukan pengecekan pada sebuah kondisi dan menentukan nilai dihasilkan Ketika kondisinya bernilai true atau false. cara penulisan (kondisi) ? benar : salah. contoh ( x % 2 == 0 ) ? "genap" : "ganjil".

Popup Box / Dialog Box

Definisi

Adalah sebuah window kecil pada browser kita yang bisa kita munculkan sebagai alat interaksi yang digunakan agar user dapat berdialog dengan halaman web. ada 3 jenis popup box yaitu ada alert, prompt, dan confirm.

Popup Box / Dialog Box

1. alert

Popup alert digunakan untuk menampilkan pesan saja, popup ini memiliki tombol ok yang jika diklik maka popupnya akan hilang. contoh penulisan

        
    alert('Penyimpanan anda penuh ! ');
2. prompt

Untuk popup prompt ini bisa mengembalikan sebuah nilai, jadi popup ini juga bisa dimasukan ke dalam sebuah variabel. contoh

        
    var nama = prompt('masukan nama anda !');

Popup prompt ini mengembalikan nilai string, jika ingin menjadikan tipe datanya ke integer maka beri sebuah fungsi dari javascript yaitu parseInt().

        
    var angka = parseInt(prompt('masukan angka !'));
3. confirm

Popup confirm ini digunakan untuk meminta konfirmasi dari user, popup ini memiliki dua tombol yaitu cancel dan ok, biasa digunakan untuk menghapus data. contoh penulisan

        
    confirm('Apakah anda yakin ? ');

Implementasi

Membuat aplikasi sederhana dengan popup box 

        
    alert('Selamat Datang ...');
    var lagi = true;

    while ( lagi === true ) {
        var nama = prompt('masukan nama anda ');
        alert('halo ' + nama);

        lagi = confirm('coba lagi');
    }
    alert('Terima Kasih ...');

Definisi

while

    while ( ulang ) {
        console.log('hello world');
        ulang = confirm('lagi?');
    }

    var nilaiAwal = 1;
    while ( nilaiAwal <= 5 ) {
        console.log('hello world');

Function Pada JavaScript

Definisi

  • Sebuah sub-program / sub-routine yang dapat dipanggil di bagian lain pada program
  • Merupakan struktur dasar pembentuk dari javascript
  • Disebut juga sebagai prosedur (kumpulan statement untuk melakukan tugas atau menghitung sebuah nilai)
  • Untuk dapat menggunakanya, kita harus membuat terlebih dahulu function tersebut, lalu memanggilnya
  • Termasuk kedalam First-Class Object

Kategori function

Ada dua kategori function berdasarkan pembuatanya

  • Built-in Function atau function yang sudah dibuatkan oleh javascript
  • User-Defined Function atau function yang kita buat sendiri

Built-in Function

  • Fungsi yang sudah disediakan / dibuat oleh javascript
  • kita tinggal memanggilnya saja
  • contoh function
    • alert()
    • prompt()
    • confirm()

User-Defined Function

  • Fungsi yang kita buat sendiri
  • Menggunakan keyword function
  • Nama function (optional)
  • Parameter / Argument
    • disimpan di dalam ()
    • boleh ada atau tidak, jika ada boleh lebih dari satu
    • dipisahkan oleh koma ( , )
  • Function body dibungkus dengan {}
  • Dapat mengembalikan nilai (return value) atau tidak

Membuat user-defined function

Ada 2 cara membuat user-defined function yaitu

Function declaration

Fungsi deklarasi (function declaration) adalah sebuah statement yang didefinisikan secara langsung di dalam program dengan menggunakan kata kunci function. Fungsi ini dapat dipanggil di seluruh bagian program setelah dideklarasikan.

        
    function jumlahDuaBilangan(a, b) {
        var total;
        total = a + b;

        return total;
    }
Function expression

Fungsi ekspresi (function expression) adalah sebuah fungsi yang didefinisikan sebagai nilai dari sebuah variabel atau konstanta. Fungsi ini hanya bisa dipanggil setelah variabel atau konstanta yang memuat fungsinya didefinisikan.

        
    var jumlahDuaBilangan = function(a, b) {
        var total;
        total = a + b;

        return total;
    }

Pemanggilan function

Setelah function dibuat selanjutnya adalah pemanggilan function

                            
    alert(jumlahDuaBilangan(2, 4));

Perbedaan declaration dan expression

Function declaration
  • Lebih flexibel (dapat ditulis dimanapun) karena konsep Hoisting
  • Mudah dipahami pemula
Function expression
  • Harus didefinisikan atau dibuat terlebih dahulu sebelum dipanggil
  • Lebih powerful, bisa dijadikan sebagai closure, bisa sebagai argumen untuk function lain, dan IIFE

Argument
Argument adalah nilai yang dikirimkan ke parameter saat fungsi dipanggil. perbedaanya adalah jika parameter digunakan ketika fungsi dibuat sedangkan argument ketika fungsi dipanggil.

Refactoring JavaScript

Definisi

Refactoring adalah sebuah proses mengubah kode agar menjadi lebih baik tanpa mengubah fungsionalitasnya.

Kelebihan refactoring

  • Readability atau kemudahan dalam membaca kode program
  • DRY (Don't Repeat Yourself) atau mencegah penggunaan kode yang berlebihan
  • Testability atau kemudahan dalam pengetesan atau pengujian kode program
  • Performance atau meningkatkan performancy kode program
  • Maintainability atau memudahkan kode program kita untuk dikelola atau dikembangkan

Refactoring sederhana

    function jumlahVolumeDuaKubus(a, b) {

        total = volumeA + volumeB;

        return total;

    alert(jumlahVolumeDuaKubus(8, 3));

Rekursif Pada JavaScript

Definisi

Rekursif adalah sebuah fungsi yang memanggil dirinya sendiri. pastikan pada penggunaan rekursif ini fungsinya harus berakhir dan menghasilkan sebuah nilai. untuk menghentikan rekursif dibutuhkan kondisi yang namanya base case atau kondisi akhir dari rekursif yang menghasilkan sebuah nilai.

Implementasi

Contoh membuat function untuk mencetak angka 10 sampai 1 

        
    function cetakAngka(n) {
        // base case
        if ( n === 0 ) {
            return;
        }
        console.log(n);
        // rekursif
        return cetakAngka(n-1);
    }
    cetakAngka(10);

Faktorial

Faktorial bisa digunakan dengan rekursif 

        
    function faktorial(n) {
        if ( n === 0 ) return 1;
        return n * faktorial(n-1);
    }
    faktorial(5);
Struktur dan hasil faktorialnya 
        
    faktorial(5)
    5 * faktorial(4)
    5 * (4 * faktorial(3))
    5 * (4 * (3 * faktorial(2)))
    5 * (4 * (3 * (2 * faktorial(1))))
    5 * (4 * (3 * (2 * 1)))
    5 * (4 * (3 * 2))
    5 * (4 * 6)
    5 * 24
    120

DOM pendahuluan

Definisi

DOM adalah singkatan dari Document Object Model, adalah representasi elemen HTML pada dokumen menjadi sebuah object. elemen HTML yang dibuat nantinya oleh browser akan di representasikan sebagai object atau dimodelkan sebagai object yang disimpan ke dalam browser.

Materi Pada DOM

Pohon Hierarki DOM / DOM Tree

Adalah representasi elemen-elemen HTML didalam browser dan representasinya dalam bentuk tree atau pohon. di dalam DOM tree semua elemenya disebut sebagai node atau simpul, tiap-tiap simpul atau node memiliki tipe yang berbeda, ada banyak tipe node

  • Elemen
  • Text
  • Document
  • ...

Istilah Dalam DOM Tree

Istilah yang harus diketahui,Node atau simpul adalah satu bagian bertipe apa saja di dalam DOM tree, NodeList adalah kumpulan node bertipe apa saja di dalam DOM tree, dan HTMLCollection adalah kumpulan node khusus bertipe elemen HTML saja.

NodeList vs HTMLCollection

  • Keduanya merupakan kumpulan node
  • Struktur datanya mirip array
  • NodeList dapat berisi node apapun, sedangkan HTMLCollection harus berisi elemen HTML
  • HTMLCollection bersifat live sedangkan NodeList tidak, penting diketahui untuk memahami DOM Manipulation

Struktur Hierarki DOM

  • Root Node
    • node yang menjadi sumber dari semua node lain di dalam DOM
    • document
  • Parent Node
    • node yang berada 1 tingkat diatas node yang lain
    • body adalah parent dari h1, p, div, a
  • Child Node
    • node yang berada 1 tingkat dibawah node yang lain
    • h1 adalah child node dari body

DOM Prevent Default

Definisi

Mencegah sebuah element melakukan fungsi defaultnya. contoh tag a yang memiliki atribut href, jika href dikosongkan maka default tag a adalah merefresh halaman, untuk mencegah hal tersebut maka kita bisa gunakan sebuah method yaitu elemen.preventDefault().

Contoh lain penerapan method tersebut antara lain ketika klik sebuah tombol, ketika melakukan checklist, memilih sebuah elemen di combobox, melakukan submit pada form, dan lainya.

        
    // HTML

    // javascript
    const close = document.querySelectorAll('.close');
    close.forEach(function(el){
        el.addEventListener('click', function(e){
            e.target.parentElement.style.display = 'none';
            e.preventDefault();
        });
    });

Event Bubbling

Ketika kita memberikan event pada sebuah elemen maka event tersebut akan berlaku untuk elemen-elemen pembungkusnya juga.

Agar eventya tidak mempengaruhi elemen pembungkusnya maka kita bisa gunakan method elemen.stopPropagation().

                                    
    // HTML

        });
    });

Abstraksi atau agar kode yang kita buat menjadi lebih sederhana, karena dengan Higher Order Function itu menyembunyikan kerumitan. Semakin besar sebuah program, semakin tinggi kompleksitasnya, semakin membingungkan programmernya.

Array.prototype.reduce()

reduce berfungsi untuk melakukan sesuatu terhadap seluruh elemen pada array, reduce meiliki 2 argument. contoh kasus newAngka adalah jumlah seluruh elemen pada array angka

    const newAngka = angka.reduce((accumulator, curentValue) => accumulator + curentValue, 0 );
0 diatas adalah optional, 0 diatas adalah nilai awal di awal array sebelum dijumlahkan, bisa diisi dengan selain 0. gambaranya adalah ( 0 + -1 + 8 + 9 + 1 + 4 + -5 + -4 + 3 + 2 + 9 )
Method Chaining

Method Chaining adalah menggabungkan fungsi fungsi pada higer order function dalam satu kali eksekusi, contoh cari angka >= 5 dengan filter, lalu kalikan 3 dengan map, dan jumlahkan semua menggunakan reduce.


    const angka = [-1, 8, 9, 1, 4, -5, -4, 3, 2, 9]; 

    const hasil = angka.filter(a => a > 5 )
        .map(a => a * 3 )
        .reduce((acc, cur) => acc + cur );
lebih besar dari 5 ( 8, 9, 9 ), dikalikan 3 ( 24, 27, 27 ), dan dijumlahkan semua ( 78 ).

QuickTip Var, Let & Const

Pembahasan

Ada 3 cara untuk membuat variabel di dalam javascript ini, ada var, let, dan const. mulai sekarang jangan menggunakan var dan ganti dengan let atau const, karena var menggunakan function scope sedangkan let dan const menggunakan block scope yang biasa dipakai di bahasa pemrograman lain.

kenapa block scope, karena normalnya pada sebuah bahasa pemrograman variabel yang dideklarasikan didalam block scope ({...}) tidak bisa diakses di luar scope, dia hanya akan berfungsi di dalam scopenya, namun keyword var ini menggunakan function scope yang membuat variabel luar bisa mengakses variabel didalam scope, ini akan mempersulit jika ada nama variabel yang sama di scope yang berbeda, jadi lebih baik gunakan let atau const.

Perbedaan Let & Const

Let digunakan untuk variabel yang nilainya bisa berubah-ubah, contohnya pada looping for. sedangkan Const digunakan apabila anda yakin nilai variabel tidak akan berubah, contohnya pada DOM selection. keyword Const juga berfungsi untuk meminimalisir perubahan state, jadi sebuah variabel itu baiknya nilainya tidak berubah-ubah karena jika kita menimpa-nimpa isi variabel dan programnya semakin besar maka akan semakin merepotkan.
